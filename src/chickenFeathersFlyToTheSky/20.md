# 第20集：陈大光自首

陈金水和江河谈起骆玉珠，认为这个女人野心太大，又只认钱，会拖累陈江河。陈江河不认同他的话，但着陈金水的病情，并不想和他争辩。谁知 ，这话，却被门口的骆玉珠听个正着。玉珠的心里十分不是滋味。连做生意也提不起精神来，转头，有听到有人在调侃为什么王旭和陈江河。玉珠顿时火冒三丈，大闹一场 。好在陈江河赶回，将她拉走。

骆玉珠气不过，转身离开，陈江河带着王旭回家 。准备进门时，却发现了失踪近一个月的陈大光在门口徘徊。陈江河百般询问，才知道，陈大光之所以发了大财，竟是做的走私生意 。这次他的货在海关被人给扣押了，以前的那些狐朋狗友们逼债的逼债，逃跑的逃跑 。陈大光被逼的走投无路，这才来找陈江河。

陈江河听说陈大光竟然做这种事，简直是气不打一处来，他要求陈大光去向公安机关自首，争取宽大处理。陈大光，担心巧姑不会等他 。

骆玉珠则熬了鸡汤，送去了医院，本来是想和陈金水和解，但两个都是倔脾气的人，说到最后，玉珠又忍不住说了陈金水几句 。但两人的关系总算是有了那么一丝丝的缓和。

陈江河和巧姑送大光去自首，公安局门口，巧姑才明白陈大光做了些什么 。深恨陈大光不争气，但依然表示会等他回来。

金土叔得知儿子被陈江河送进了公安局，气的打上门来，还口不择言的诬陷巧姑和陈江河。骆玉珠听到这话，忍不住冲上前去，打了金土一个耳光 。江河忙将玉珠拉走。

晚上，江河劝玉珠去向金土道歉，玉珠坚决不去，两人起了争执 。玉珠气的转身进屋，江河说服不了玉珠，好在王旭借口要给玉珠敷药，给了江河一个台阶 。江河趁机服软 。两人重归于好 。

金水听说玉珠打了金土一个耳光，觉得十分解气，连声说打的好 。玉珠和江河来接金水出院 。金水要还他垫付的住院费，江河不要，说就是因为金水叔的一句积少成多，才让他想到了走批发这条路 。江河提出要请巧姑去店里帮忙 。在江河的撺掇下，玉珠向金水叔道歉，还邀请金水叔做他们的顾问，虽然金水叔没有马上同意。但显然，金水叔对于陈江河想到的批发这条路十分认同，还颇为自豪。

后来的日子里，江河和玉珠的生意越做越大，生意十分红火 。还买了一台卡车 。专门用来运送货物 。

一天下大雨，玉珠正在家里整理货物等待发货 。陈江河从外面来，脸色十分不好看。原来他们这批货赶着发走，但货运站的站长，却百般刁难。陈江河十分发愁 。
