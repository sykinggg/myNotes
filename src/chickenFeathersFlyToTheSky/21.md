# 第21集：江河获得小旭信任

陈江河的货物着急发货，货运站的站长趁机吃拿卡要，骆玉珠提议送个红包，陈江河不同意 。但货物发不走，他们就会面临违约，损失严重。陈江河最终无奈选择送礼 。

陈江河送了一盒茶叶给站长，站长打开一看，里面竟是一双破鞋，站长十分生气，把鞋扔了出去，却被门外人捡起 ，原来正是新上任的改革办主任邱英杰 。邱英杰多次听说这个货运站存在问题，正是来考察的。结果正遇上，陈江河在买茶叶，于是趁机将茶叶换成了鞋子 。这双鞋的主人曾穿着它走了几十里的山路挑着扁担鸡毛换糖。邱英杰准备将这双鞋送到义乌的博物馆里 。货运站站长免职，门口等待的商贩们十分高兴。陈江河却注意到邱英杰的脸色十分苍白。忙问他要不要去医院 。邱英杰却说没事。只是，因为马上要出国考察，准备将女儿邱岩送到陈江河家住几天 。江河自然一口答应，他们两口子都很喜欢邱岩这孩子 。

邱岩聪明又懂事，她在陈家，主动帮玉珠干活，还帮助辅导王旭的作业，相处的十分愉快 。玉珠看着两个孩子的背影，显然有了更长远的想法。

玉珠发货给零售商，清点后发现有一箱少了一件，玉珠大方表示，所有的货物都按照九十九件来算 。路上，玉珠百思不得其解，但江河却表扬玉珠这件事情处理的很好 。

玉珠正在教巧姑怎么摆放货物，那天拿货的商家又带着两个人上门，玉珠还以为是货物还有差错，忙笑脸迎上去。却原来是那位客户觉得玉珠做生意十分大气，又给她推荐了两位客户，玉珠一下子卖掉了不少 。心里十分高兴。

邱岩在陈家和王旭相处的十分愉快，王旭还偷偷拿玉珠的发卡送个邱岩，玉珠看到，只感叹这小子，讨着了媳妇气死老娘。王旭作文得了第一名，邱岩高兴的告诉陈江河夫妇，但没想到王旭写的是他的爸爸王大山 。

晚上，江河主动和王旭谈起王大山，两人在沟通中渐渐拉近了距离 。

热水器接连被人退货，江河着急，赶忙带着机器去找夏厂长，查看是否是质量问题。就在江河出去的一段时间里，热水器又接连被人退回来两台 。陈江河回来后，却说，这机器并不是他们店里的，而是被人假冒的。只可惜外形一摸一样 。他们也无法辨别 。

晚上，玉珠在灯下写写画画。原来，玉珠聪明的想到了自制防伪标记 。她带领两个孩子将赶制出来的防伪标记塞进了箱子里 。

巧姑和江河去监狱看望陈大光，陈大光因为立功表现，被减刑，大光向巧姑承认错误，巧姑则表示自己会等他出来。大光感叹自己早该听从江河的建议 。

玉珠又遇到来退货的人，却拿不出玉珠手制的质量保证书，那人尴尬离开。玉珠偷偷跟在后面，原来是金土在背后捣鬼。被玉珠抓个正着。

江河和巧姑回来，见到在店里招呼的是金土，玉珠则坐在一旁休息。十分惊奇。
