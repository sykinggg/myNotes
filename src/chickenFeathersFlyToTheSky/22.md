# 第22集：假冒伪劣商品风波

江河和巧姑见大光爹在卖货，十分惊讶，忙将他劝走。晚上，江河劝玉珠不要这样，玉珠告诉她，那些假的热水器就是大光爹在背后捣的鬼 。

江河去金土家打听那些制造假货的人的消息。一进门，真巧遇见，有人要强行搬走金土叔家的电视机，说是金土欠了他们五百元钱，换不了 。陈江河让他们放下电视机，替金土叔还上了钱。金土十分后悔自己的所作所为 。但他也不知道那些制假的人在哪里，只知道他们什么好卖做什么 。

玉珠有意调教巧姑，让巧姑自己去和顾客谈价，但巧姑性格太软，有没有什么主见 。看的玉珠十分着急，只有自己亲自出手，巧姑十分钦佩，玉珠趁机教育巧姑，拿金水叔当年鸡毛换糖的经验举例子 。

江河带金土回来，让玉珠给金土一批货拿去零售，卖的利润再用来还陈江河的五百块钱 。玉珠本不情愿，但江河却说不然连五百块都收不回，玉珠无奈同意 。

巧姑将玉珠的话学给金水听，金水听的十分受用，认为玉珠倒是有肚量 。

随着改革开放的深入，人们开始追求各种国外的名牌，于是，义乌的仿冒之风兴起，各家商户都称自己的东西是国外进口的，由于价格低廉，又号称是名牌 。生意十分火爆 。而只有玉珠一家，由于陈江河之前立下的规矩，生意一下子变得十分清淡，而且利润也无法和那些假冒的商品相比。玉珠对此十分发愁 。就连金土叔也退回了江河给他的货物，说是要去进一些好卖的货 。玉珠听了 ，更是心里一团乱麻。

而当时的经济形势下，三角债的情况十分严重 ，上中下游三方互相欠钱，最下游的零售商东西卖不出去，还不了钱，同时中间商又欠着上游厂家的货款 。而陈江河多年来坚持不欠别人的钱 。但借出去的钱又都收不回来 。一下子让自己的资金变得十分紧张 。

玉珠接到电话，有人要给他介绍一批好卖的货，玉珠到了才知道，原来是仿造国外名牌的箱包 。玉珠本来坚持不要，但看到有好几家的老板都在等着进货 。竟然一咬牙，将所有的钱都投了进去，购买了一大批假冒箱包 。

玉珠刚刚将货物运回家，陈江河回来了，还带着刚从国外回来的邱英杰 。玉珠一下子慌了神 。

邱英杰还带回来一个消息，义乌要开始整顿市场，狠狠打击假冒伪劣商品 。同时也要开始整治三角债的问题。玉珠听到这个消息，仿佛一个炸雷在耳边炸响。陈江河从进门开始就觉得玉珠不对劲 。送走邱英杰和邱岩后，陈江河才知道，玉珠将手上所有的现金都拿去进了假货 。陈江河气急却不知说什么好，只能说，就当这批货没有进过，不能拿出去卖 。

义乌市场果然开始打击假冒伪劣，每天都有很多人的货物被收缴。玉珠则每天都魂不守舍 。

陈江河没有了办法，只有去求邱英杰 。见到邱英杰正忙着整治假货，同时还在联系银行，希望能帮助商户解决三角债务，帮他们渡过难关 。陈江河实在不知道该如何开口 。邱英杰听说了陈江河的来意，好好的给陈江河上了一课，告诉他，商人最重要的不是赚钱，而是信任 。
