# 第23集：邱英杰积劳成疾

江河来找邱英杰想办法，邱英杰是真没想到，陈江河也会头脑发热做这种事情，邱英杰拒绝了陈江河，并跟陈江河好好的上了一课，告诉他商人不能只想着赚钱，而是要想着信用二字。如果没有了信用，商户之间的三角债将会成为一个巨大的隐患。很快，邱英杰说的话实现了，由于没有了流动资金，下游的零售商拖欠的货款又收不回来，一向不欠钱的的陈江河也尝到了被债主上门要债的滋味。

骆玉珠这边，由于好几天都觉得恶心，到小诊所去开了消炎药又不见好转。玉珠终于到正规医院检查了一次，结果却发现自己竟然是怀孕而不是胃病。玉珠气的冲到小诊所去砸了医生的桌子。晚上回家，陈江河正在为生意和资金的事情犯愁，玉珠思量再三，终于还是哭着告诉江河，她怀孕却因为误听了黑诊所庸医的话服用了抗生素，这个孩子可能保不住的消息。陈江河目瞪口呆，转身摔上车门离开。

几件坏消息接踵而来，陈江河心情十分郁闷，只能出来借酒浇愁 。却听说有人状告邱英杰官商相互，而这个商，说的正是他陈江河。陈江河醉醺醺的来到邱家，竟发现有人趁夜往邱家扔砖头，还砸碎了玻璃 。陈江河再次请求邱英杰帮忙，依然被拒绝，陈江河心里不痛快，询问邱英杰是不是因为有人告他，所以才不能帮他。邱英杰万万没想到陈江河竟然说出这种话来，气的将陈江河大骂一顿 。

第二天 ，陈江河酒醒了，虽然他很希望有个孩子，但还是尊重玉珠的意见，选择打掉。但两人这几天的相处颇有些尴尬 。二人来到市场，发现好多人为了要回被扣押的货物，竟然集体到邱英杰办公室门口闹事去了。江河听了，转身就走。英杰办公室门口已经被商户们围了起来，邱英杰走出来，举着手上的数据，苦口婆心的劝告大家，卖假货不是长久之计，唯有踏踏实实的做生意，打响自己的品牌，才是正道，混在人群中的陈江河深受振动 ，众人也理解了邱英杰的做法，纷纷散去 。

陈金水也来了，他特意等在门口，拉邱英杰去他家下棋，只可惜邱英杰实在是没有心情。他既要忙着跑银行，帮助商户们度过三角债的危机，又要和贩卖假货的商户们打游击。金水叔提出让他和几个乡镇的退休的老干部们代替他去整顿市场 。邱英杰听了十分感激 。

陈金水来到义乌市场巡查，商户们见到他，纷纷将假货收了起来 。陈江河看到金水叔，主动上前，请金水叔检查，并保证自家不会有一件假货。陈金水听后十分欣慰 。

陈江河和玉珠之间的尴尬气氛连王旭都看出来了，王旭以为是家里没有钱了，拿出了自己的储蓄罐，希望他们能和好，看的玉珠又心酸又骄傲。

江河陪玉珠去医院做手术，谁知却意外得到一个好消息，原来玉珠吃的所谓消炎药，也是假药，成份是淀粉。得知孩子没有问题，二人悬着的心终于放了下来，两人相拥 。

这边，邱英杰也在医院检查，却得到了自己的癌细胞已经扩散，恐怕不久于人世的消息。
