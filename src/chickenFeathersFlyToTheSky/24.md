# 第24集：江河玉珠智擒大狗

邱英杰得知自己时日无多，在夜晚来到空无一人的义乌市场，看着这个自己一手打造起来的市场，依依不舍。灯下，邱英杰不顾自己的身体，依然伏案工作。他要在自己离开之前，做一件大事 。

陈江河夫妇得知孩子没有问题，十分高兴。江河特意下厨做了几个好菜，王旭十分稀奇，问什么是添丁 。却被玉珠糊弄了过去。可是，王旭第二天就从邱岩的口中得知了添丁的意思，知道玉珠又怀孕了，当时便变了脸色 。

王旭回到家里，便一言不发，不管谁跟他说什么都不说话陈江河猜测他大概是知道了。想要想办法跟孩子沟通一下，玉珠却说，只要能真的做到一碗水端平，自然就会好的。

江河去找邱英杰，竟然听说英杰辞职了。十分惊讶，四处打听才知道，货运站站长联合几个商户向上级写检举信，说这样可以把货要回来 。邱英杰肯定是因为这才辞职的。江河十分气氛，决定以其人之道，还治其人之身。他同样搜集了更多人的联名检举信，当着站长的面，将检举信递交给了政府 。

陈江河和骆玉珠刚刚回到市场，就金土叔说他找到了那几个造假热水器的人。原来，冯姐发现有人在卖和她家一样的衣服，特意请金土叔盯着点，谁知金土叔竟然发现正是当初造假热水器的那一伙人 。带头的是个年青人，叫大狗 。

玉珠和江河忙带着金土叔一起去找这个大狗，他们借口有一百多万的大单要做 。故意引诱大狗上钩，玉珠，江河和金土叔三个人一唱一和，配合默契，将大狗唬的一愣一愣的。最终终于上了钩 。

大狗带他们去了自己的窝点，像他们展示自己造假的““实力““ 。玉珠借机要求他们先做五千件首饰给他们看看，大狗却说，他们马上就要转移了。江河灵机一动，乘机提出了让他们在自己卡车的后面现场制作假首饰 。在重金的诱惑下，大狗欣然同意。谁知，陈江河竟把车直接开进了工商局 。

抓住了这个制假团伙，众人都觉得大快人心，邱英杰得知这个消息，笑着感叹，还是陈江河有办法 。

陈江河在邱英杰家收了两个多小时，终于等到邱英杰回来，他们一起走在乡间的小路上，聊起了义乌的前景，还聊起了他们当年相遇的往事。心中感慨万千。邱英杰将他收藏多年的那张车票送给了陈江河。陈江河看着邱英杰苍白的脸色，心里十分放心不下。邱英杰却安慰他没事。

邱英杰抱病请来各家商户和欠债的零售商，希望能将义乌商户手里的债务问题解决 。会上 ，双方再次吵成一团 。
