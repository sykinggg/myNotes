# 第27集：玉珠慧眼识商机 路上生子

骆玉珠回到家里，却发现王旭没回来，陈江河告诉玉珠，他到学校时，就听说王旭和邱岩一起离开了。正说着，邱岩妈妈找上门来，询问邱岩的下落。这才知道两个孩子都没回家。大家急忙出去寻找。骆玉珠突然想到当年的事情，猜测王旭会不会带着邱岩回赣州去了。二人沿着铁路线寻去。整整一天，终于找到了两个孩子。

陈江河又气又急。忍不住把王旭拉过来要打他。王旭知道错了，开口求饶，终于喊了声爸爸。陈江河楞在当场，哪里还打的下去。

邱岩最终还是跟着母亲出国了。临别前，两个孩子依依惜别。

骆玉珠八个月了，她到医院产检时，见识了叫做尿不湿的新鲜玩意。从中见到了巨大的商机。回来便决定要做尿不湿的生意。但当时，尿不湿还是个新鲜玩意，没有多少人用。所以不仅是陈金水不同意，连陈江河也不支持她。骆玉珠生气离开。

回到店铺，玉珠一直想办法到处打电话，希望能联系到厂家。可是一直没能得到厂家的联系方式。另外一边，正在跟人谈生意的陈江河，突然注意到旁边有人再给孩子使用尿不湿。忙凑过去打听。

玉珠终于打听到消息，不顾自己的身体，也要亲自去拿下这单生意。巧姑阻拦不住，只有跟着一起去。两人来到工厂，玉珠顺利说服了厂长，将货物全部给他们分销，只是说到要先赊欠时，厂长本不愿意，后来听说他们是银珠五金的。厂长欣然同意。原来，玉珠和江河在宁愿卖车也要还清欠款的事情已经传开了，大家都相信他们的人品。

陈江河听说骆玉珠亲自去进货，吓了一跳，忙骑车去寻他们，却在高速路口被拦下。这边，骆玉珠突然觉得肚子剧痛，居然是要生了。巧姑在一旁慌了神，不知如何是好。

陈江河只有折返，刚到家就接到了巧姑的电话，原来她们已经到医院了。孩子已经出生了。陈江河听到孩子的哭声，半天都回不过神来。这个孩子，被骆玉珠取名陈路。

转眼到了2005年，陈路小小年纪就展现了过人的计算机天赋。而王旭在大学里也不忘做生意，施展他的生意头脑。陈江河的生意也越做越大。

骆玉珠和四个西班牙商人谈判，价格一直谈不拢。骆玉珠转身离开，留下四个老外面面相觑。

而陈江河这边却不顺利，他到欧洲约见他的生意伙伴费尔南德，但对方却拒绝见面。他怀疑对方又走了新的合作伙伴。

骆玉珠请王旭找了几个西班牙同学帮忙翻译一段录音。原来那四个商人早就合谋，想要压低价格。只有一个人提出过异议。骆玉珠于是亲自找上门去。与这个叫莱昂的人单独谈合作。

陈江河偷偷跟在后面，想要看看是谁在捣鬼。没想到却见到一个熟悉的人，杨雪。

骆玉珠和莱昂详谈后，莱昂十分敬佩江河和玉珠。玉珠邀请他参观他们的工厂。

陈江河费尽了周折，终于见到了费尔南德，陈江河好言相商，费尔南德却趾高气昂。还说他愿意按照合约赔偿陈江河五万美元的损失。但陈江河的货物滞留港口一天的损失，就是五万欧。

陈江河回到酒店，绞尽脑汁也想不到好办法，却突然接到杨雪的电话。约他去她住的酒店见面。
