# 第28集：杨雪使计 邱岩回国

杨雪约陈江河见面，两人多年不见，现在相见，却颇为尴尬。杨雪借酒劲，向陈江河倾诉这些年她的孤单和怨恨。陈江河没想到杨雪越来越偏激。他知道费尔南德之所以违约，就是因为杨雪从中作梗。她想要将玉珠集团打垮。

杨雪开出条件，要陈江河陪她一夜。好让新闻传回国内骆玉珠的耳朵里，要么就让陈江河带着自己的货回国去。但这样的损失，陈江河也是承受不起的。陈江河看着杨雪觉得十分陌生。

国内骆玉珠一直在关心陈江河和欧洲的货物的状况。但一直打不通电话。陈金水告诉骆玉珠他们的产品很快就有了仿冒品，还几乎可以以假乱真。还告诉玉珠有几个小厂，因为要达到欧洲的生产标准，产量下降，联名写了信，送到了陈金水这里来。玉珠劝说陈金水好好休息，公司的事情有她和江河操心，陈金水认为玉珠是嫌弃自己老了。转身离开，两人不欢而散。骆玉珠让巧姑调出了这几天的监控，发现了一个可疑的男人。玉珠却觉得那个人十分眼熟。

陈江河打电话来告诉玉珠欧洲这边的情况时，玉珠正在和几个厂长吃盒饭，几个厂长想要回到以前的生产标准和产量。却被玉珠无视。

陈大光忍耐不住告诉陈金水欧洲的客户被杨氏抢走了，货物也滞留在港口。陈金水十分担心，但仍然嘱咐陈大光不准让消息走漏出去。

陈江河发现杨雪的货物也一直放在港口，就连费尔南德的人来都提不到货。觉得很奇怪，难道是杨雪自己愿意出货？可是这是为什么呢？

玉珠集团，展销会就要举行，陈金水来找骆玉珠询问欧洲的事情，骆玉珠只说他相信陈江河的智慧，希望大家能同舟共济，共度难关。

玉珠派司机从机场接回了一个年青女孩。正是当年的小邱岩。

展销会上，来的都是各国外商。骆玉珠不信任他们自带的翻译，于是王旭从学校里请了他的几个同学临时充当翻译的角色。没想到展销会就要开始了，翻译们却还被堵在路上。骆玉珠只好找人给他们表演节目，拖延时间。

王旭他们终于到了，不过，当他们赶到时，发现那些外国商人已经开始参观工厂了。陪同的竟是一个年青女孩。她一个人居然搞定了来自世界各国，讲不同语言的商人。王旭和邱岩在分开了近十年后，终于重逢了。

骆玉珠问莱昂，是否考虑好跟她们合作，莱昂十分为难。一方面他很想跟玉珠集团做生意，另一方面，他和他的朋友们都不敢得罪欧洲巨商费尔南德。骆玉珠十分失望，说原来莱昂是狮子的意思，他却是只温顺的狮子。
