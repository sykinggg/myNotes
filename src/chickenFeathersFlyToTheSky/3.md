# 第3集：骆玉珠陈江河再聚首

骆玉珠听说鸡毛要回来了，内心百感交集。

义乌市国营纺织厂仓库门口，几位工人正在盘点仓库里的不要的棉布头，一个年轻人走了进去。正是三年不见的陈江河。他听说纺织厂要将废弃的棉布头当垃圾处理掉。赶上前去劝说厂长将棉布头卖给他。并出示了他亲手抄写的政府关于鼓励经营的通告。厂长有些动心，但仍然没有一口答应，只说要商量一下，要陈江河回去等消息。

鸡毛回到陈家村，整个村的老少爷们都惊动了。鸡毛看着率先迎出来的大光，踌躇着不敢相认。片刻后，这对儿时的玩伴紧紧地拥抱在一起。

鸡毛看着从人群中走出的陈金水。眼中含泪，跪倒在地。陈金水扶起鸡毛，爷俩相拥而泣。晚上，一家人终于吃上了一顿团圆饭。

鸡毛给陈家村带了一台电视机，这可是村民们从来没有见过的新鲜玩意儿。全村老少都聚在一起凑热闹。正当大家看得起劲儿时，陈金水悄悄的起身离开。

鸡毛见到，赶紧跟了出去，原来陈金水正躲在一旁抽着闷烟，心事重重。看到鸡毛跟来，问起鸡毛在外吃过的苦。鸡毛毫不在意的一笑。陈金水却满脸心疼。鸡毛准备跟金水叔讲讲他下一步的计划，金水却要劝他快逃走。鸡毛十分不解，认为现在都在鼓励放开手脚搞经营，自己合法做生意为什么要逃。

陈江河接到消息，国营纺织厂终于答应将这批棉布头卖给他，但是同时又另一拨人也要买布而且今天就要拖走。陈金水忙让大光开着村里的拖拉机，赶往国营纺织厂。

江河来到纺织厂，货已经被拖走了，厂长说是几个女同志，死缠烂打的非要拖走。江河急忙掉头，追了出去。

小路上，几个女人推着堆满了棉布头的板车，艰难前行。陈江河追了上来，冲上去要她们留下这车货。谁知这车正是骆玉珠的。两人相见，悲喜交加。

两人相见的地方，正是当年他们初遇的那个桥洞。两人站在桥洞下，回忆起过去。陈江河十分欢喜。骆玉珠看着陈江河，却似乎另有心事。

两人谈好再次合伙做生意，五五分成。看着两个人谈起生意一本正经的样子，到让大光和冯姐摸不着头脑，不知道这两个到底是熟还是不熟。

原来陈江河买来棉布头，是要组织村里的年轻人做拖把。没想到被柱子叔和大光爹发现了，出人意料的是，他们听说了陈江河的计划，居然没有阻止，反而认为陈江河跟骆玉珠五五分是没有生意头脑。自己卖明明可以赚的更多。

晚上，陈江河和骆玉珠在桥头见面，这时，骆玉珠才知道，陈江河是打算把这次挣的钱全部分给陈家村的乡亲们，因为他漂泊多年后才明白陈家村是他的家。

骆玉珠听陈江河说起他在外时很是想念一个人时，低头默默不语。谁知陈江河说的却是邱英杰。玉珠气的推了他一把，转身离开。陈江河十分莫名。又转头听陈江河脱口而出的忘不了她当初骗的他团团转。又迅速转怒为喜。

1992年，县里开办学习班。召集陈金水等基层干部封闭学习。老师正是邱英杰。各位干部看到邱英杰，脸色十分精彩。邱英杰讲的第一课，正是解放思想。

仅仅三天的时间，通过邱英杰升入浅出的讲解，陈金水的思想明显松动了不少。邱英杰带来的北京的消息，终于让陈金水放下压在心头多年的大石头，他终于认识到，鼓励经营是中央的政策。正经做生意的人再也不会像他当年一样被当成投机倒把而被抓走了。
