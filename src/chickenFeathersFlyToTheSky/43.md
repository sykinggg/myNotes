# 第43集：陈江河壮士断腕出困境

陈金水听说两人分居，忙赶往骆玉珠居住的酒店劝说骆玉珠，说起当年他的阻拦和二人经历的艰难。好说歹说，终于将玉珠劝了回去。陈江河盛装等在门口，专门等着玉珠回家。晚上，玉珠故意躺在沙发上睡着。陈江河也陪着她躺在沙发上，早上二人都哈欠连天。

欧洲，费尔南德在电视上指出一个来自中国的名字叫“王珠”的品牌铅含量超标。他意图以此说明中国产品的质量达不到标准。

欧盟出台新的标准，同时专门对中国的首饰五金产品展开调查，此消息一出，各方震动。连远在山区的王旭和邱岩都当即决定回来。

陈江河在办公室关了一整天，最终决定壮士断腕。召回欧洲所有商品，工厂全面停工停产。直到找到最新的符合标准的材料为止。骆玉珠却不同意，这样做玉珠集团将会蒙受巨大的经济损失。两人意见不同，再次起了冲突。

王旭和小玉告别，小玉十分不舍，王旭安慰小玉不要哭，要坚强。邱岩看出费尔南德用的是借龟拒虾的招数，王旭受到启发，想到了危机公关的应对策略。他给骆玉珠打电话，得知他和陈江河想到了一起。

邱岩回公司组织召开新闻发布会，王旭则赶往各大工厂，组织了一次大型的义务献血的活动。一方面证明玉珠集团的员工并没有铅中毒的情况发生，另一方面，也可以提升玉珠集团的企业形象。

杨雪很欣赏邱岩，想要将邱岩收到麾下。却被邱岩拒绝。

阮文雄对杨雪很感兴趣，他和杨雪签订了大额订单。帮助杨雪向董事会交差。

邱岩的鼻炎发作，王旭听她提起她在美国的经历。坚持要带她去看病。邱岩心里十分感动。
