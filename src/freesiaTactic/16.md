# 第16集：青苍的七情树发芽

小小的村子却十分热闹，女人们有的背着弓箭，拿着长矛，有的来来往往干着家务活。一位大姐瞧见小兰花手上的花，便告诉她这花不能随意乱戴。这花名叫霜盐花，在部落里，只有寡妇才会戴。小兰花看到大家头上都戴着霜盐花，便明了整个村子都是寡妇。一旁的东方青苍似乎听到了她们的对话，低着头像是在想些什么。

一位老婆婆眼尖认出了东方青苍，随后众人纷纷跪下行礼拜见，只有老婆婆依旧坐着。婆婆询问东方青苍，这三万年他究竟去了哪里。这三万年，苍盐海沦为废墟，族人受尽苦楚，家家户户流离失所，丧失亲人，可月族月尊却不知去向。婆婆说着说着便哭了，他的儿子原本应该和东方青苍一个年纪，可是如今东方青苍回来了，他的儿子却没有回来。婆婆请求东方青苍，他当初是怎么把那些男人带出去的，就把他们带回来。东方青苍盯着婆婆，可却没有给出回答。

小兰花跟着东方青苍来到一处悬崖，耐心劝慰他。东方青苍不由得想到了自己的小时候，那时父尊会亲自教导他练剑修法，会宽容他的调皮，会和他一起踢蹴鞠，似乎日子就应该是这样快乐的。想到这里，东方青苍脸上露出了不一样的神色，小兰花还是第一次在东方青苍的脸上看到这种表情，不免感到有些有趣。

刚到苍盐海时，所有人都说月尊的地位至高无上，任何人以任何理由冒犯了东方青苍，都是死路一条。可是刚刚，那个老婆婆那样训斥东方青苍，他都没有为难，小兰花觉得定是老婆婆对亲人的感情和牵挂打动了他。东方青苍坚决否认，直言自己只是不忍她一个年老将死之人罢了。可这不忍也是七情中的同情和悲悯，东方青苍明明七情尽断，怎还会生出这样的情绪，小兰花一时间想不明白。想来想去，也只有一个答案。

两人随后来到东方青苍的心海，茫茫大雪中的七情树果真发芽了。小兰花欢呼雀跃，可是一旁的东方青苍面色凝重，似乎是在担忧什么。回去之后，东方青苍就梦到了父尊。只是父尊不再温柔，而是变得严厉，令人畏惧。东方青苍从噩梦中醒来，久久无法恢复平静。

若是感受到七情，业火也将消失，那时候的东方青苍便不再是三界最强。可是东方青苍身上背负着整个苍盐海，他不能容许这种情况出现。此时的他无法控制自己的情绪，将所有怒火发泄到小兰花身上，可他自己的心中也难受万分。谁料结黎正撞到了东方青苍的枪口上，竟在这时偷盗财宝，准备出逃。结黎谎称自己拿这些东西是为了讨小兰花的欢心，这才逃过一劫。

就在结黎长舒一口气时，东方青苍在包裹中发现了一个木雕，眼中瞬间升起杀意。在父尊变得严厉时，小青苍常常忍不住委屈落泪。一位女子拿着一个未完成的木雕过来，和他说起了一个故事。这木雕雕的是第一代月尊盐女，她带领着月族祖先渡忘川，在苍盐海扎根，从此生生不息。她的身上有很多伤痕，每一道都是光荣的见证。小青苍忽然想明白了，若想成为月尊，保护千万月族子民，就要承受伤痕。

殇阙看出东方青苍脸上的异样，设法救走结黎，可是他低估了东方青苍的狠辣。就在大刀砍下的一瞬间，小兰花忽然闪出，挡在了结黎身前。东方青苍急忙停下，可却发现自己手腕上多了一道伤。小兰花死活不肯让开，惹得东方青苍怒火中烧，青筋暴起，眼中的杀气逼得人快要烧起来。幸而有人此时来禀告，说巽风吵着要去祭奠老月尊。

虽然怨恨哥哥，可巽风也不容许苍盐海受到伤害。他告诉东方青苍，海市主并未告知自己祟气何来。东方青苍本不想重罚巽风，可是他却出言不逊。被惹怒的东方青苍哪里还顾得上什么兄弟之情，下令将巽风雷刑处死。殇阙提醒东方青苍，三日后便是老月尊的忌日。可谁料东方青苍根本不在意，甚至要将忌日提前，好成全巽风所谓的父子情深。

得知东方青苍要将巽风处死，小兰花赶忙去找他。小兰花替巽风求情，又问到了老月尊。怎料东方青苍突然发怒，控诉父尊对他的折磨。而当初老月尊让东方青苍杀了自己，以生命为代价毁掉的七情树，如今却又要生根发芽。可小兰花只觉得事有蹊跷，老月尊绝不会不爱自己的儿子，这苍盐海和东方青苍似乎并不像众人眼中的样子，这中间一定有什么隐情。
