# 第28集：小兰花向青苍求婚

若放在以前，只要是蝶衣的吩咐，她一定照办。这一次，她却犹豫了。可为了活命，她不得不答应蝶衣。见殇阙过来，结黎只好谎称自己在藏宝贝。不料殇阙不仅不怪罪，还生出了心疼之意。他搂住结黎的肩膀，承诺自己以后一定会保护她，就算她欺骗自己，自己也心甘情愿。结黎虽然不愿伤害小兰花，可她自投入海市主门下起就已中毒，受人控制，自然只能遵从蝶衣吩咐办事，将血滴入小兰花骨兰中。

东方青苍只是想去赴小兰花山月节之约，可是却遭到了族人的反对。族人们乌泱泱一片跪倒在殿门外，请求东方青苍移驾回宫。东方青苍使出业火，这才让众人安静下来。他向众人宣布，自己钟情于小兰花，不仅要与她共赴山月之约，还要为她准备一场三界中最盛大的婚礼。

众人只知道小兰花是仙族之人，可却不知她温柔如水，从未有过仙月成见，把月族也同样当做心中的苍生。众人都说她是红颜祸水，却都不知她对东方青苍的救命之恩。众人只见她的柔弱，却不知她的英勇无畏，是三界之中最有胆识的女子。东方青苍自认没有能够寻回十万将士是自己的错，但他已认定小兰花是自己唯一的妻子，所以甘愿承受噬骨霜盐钉之刑。

圆月高悬，山风吹拂。小兰花等了许久，东方青苍终于姗姗来迟。她高兴的扑上去拥抱亲吻，却不知东方青苍刚刚经历了一场什么样的刑罚。小兰花刚转身，东方青苍竟轰然倒地，没了意识。从殇阙那里，小兰花才得知，东方青苍正在经历着噬骨霜盐钉之痛。有心疼，有愧疚，有担忧，可是小兰花却没有办法替心爱之人承担哪怕一丝丝痛苦，而这也令她更加痛苦。

不知道过了多久，小兰花靠在殿前昏昏沉沉睡去。醒来时，东方青苍已经恢复如初，随后东方青苍将准备的聘礼和灵宝阁钥匙交给小兰花。东方青苍和小兰花都有话要和对方说，也都觉得自己说的话更重要。小兰花无奈，只好拉过东方青苍的手，将自己亲手编的手环放到他的掌中。她询问东方青苍，愿不愿意和自己成亲，做自己的丈夫。她向东方青苍承诺，自己一定好好对他，一辈子只喜欢他一个人，无论发生什么，都会一直陪着他。

东方青苍紧紧握着手环，久久没有回答。小兰花还以为他不愿意，眼里的光逐渐黯淡下去。见东方青苍戴上手环，小兰花终于明白他的意思。小兰花抚摸着东方青苍的手，踮起脚尖吻了上去。可巽风却不合时宜的突然出现，极力阻止这场婚事。

如果说两个人在一起一定要付出什么代价，那么小兰花甘愿成为那个付出的人。可东方青苍清楚，自古以来就没有人能从弗居洞里活着出来，所以他哪里能够容许小兰花去冒险。对于小兰花而言，想要的只有相互尊重，为了守护这份爱，她选择承担自己的责任，也希望东方青苍能够尊重自己的选择。

为了隐藏同心咒的功效，小兰花悄悄向巽风寻来灵药。见小兰花如此执着，巽风坦言自己只是想让她知难而退，并非真的想让她去弗居洞里送死。小兰花当然知道巽风爱护东方青苍，可是每个人都要为爱一个人承担责任，付出代价。此次弗居洞一行，她已下定决心。

临别时，小兰花将一杯酒递给东方青苍，希望他能够喝下。东方青苍似乎是察觉到什么，可他答应过小兰花，一切都会尊重她的选择，所以他当着小兰花的面将那杯酒饮尽。
