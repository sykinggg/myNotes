# 第3集：小兰花错会青苍感情

就在小兰花快要掉进熊熊烈火之中时，不知什么力量拖住了她。与此同时，天边的月亮慢慢变得圆润，圆的几乎快要将人吞噬。留芳阁的人赶忙掏出武器，可是那些武器却在手中化为灰烬。就在他们转身逃跑的时候，身后的火焰将他们生生吞灭。只见那圆月之中出现一个人影，将小兰花召了过去。小兰花忽然感觉自己变小了，只能任由东方青苍捏在手中。

蝶衣不清楚来人是谁，可也不容许有人破坏她的大计。可无奈东方青苍法力无边，只一个充满杀气的眼神，那些小兵们便灰飞烟灭，就连她自己也身受重伤。她还想探寻对方的身份，可只见一团祟气将她团团包裹。随后，那混黑的祟气充斥着整个海市，人们张牙舞爪，像是变成了怪物。

东方青苍将小兰花放出，用尖锐的爪子抚摸着她娇嫩的脸庞。小兰花以为他要杀人灭口，急忙要走。可没想到对方竟说她是他的，他不容许任何人伤害她。小兰花春心萌动，不由得以为这是东方青苍对自己的告白。可小兰花实在是不喜欢这个大魔头，只好委婉的拒绝了他的心意。

随后，小兰花才知晓，原来东方青苍身上有息兰圣印。可她也只是在书上看到过这个圣印，至于如何解开，她也束手无策。东方青苍似乎并不相信，可当他抽出小兰花的仙根时，他这才罢休。小兰花生气东方青苍窥探了她的隐私，气鼓鼓的就要走，殊不知自己刚刚花大价钱买回的灵药掉在了东方青苍的手上。

海市里危机重重，充满着祟气和怪物。天上忽然降下一位带着面具的男子，只见他一扬手，那些怪物便没了动静。随后他来到化魂墟，恢复了这里的样貌。此时，那些怪物已经变回了人，拍拍身上的污迹就若无其事的走了。原来面具男子正是海市主，蝶衣的主上。

自太一生水，鸿蒙初开以来，息兰一族偏居息山一隅，为三界镇守凶神太岁。他们向来保持中立，从不参与纷争，若进犯息兰，恐有失道义。可东方青苍纵横三界所向披靡，如何肯将性命寄托在区区一介小花精身上。话音刚落，他的眼睛不自觉流出眼泪，气得快要暴跳。

东方青苍悄悄潜入司命殿，找到了正泪眼汪汪的小兰花。小兰花担心有人看到自己和罪仙在一起，赶忙将他推出去。可小兰花实在不是东方青苍的对手，只能任由他大摇大摆走进殿内。两人争执之时，东方青苍发现了小兰花藏在身上的命簿。小兰花想要抢回来，可是无奈竟被他一个响指弄不见了。东方青苍窥探命簿，竟发现了赤地女子。小兰花赶忙回去抢命簿，可东方青苍也不肯放手。两人争抢着，竟烧坏了那命簿。

五百年前，长珩身受重伤闯入了一片林子里。在他失去意识的一刻，看到了一位温柔的女子。手下前来禀报，长珩的思绪这才回来。自从昊天塔异动，这水云天上上下下都查过了，可唯有司命殿还未寻过。长珩亲自前往司命殿，可却并未发现异常。此时的小兰花哭的不能自已，东方青苍恼怒可又无可奈何。试了许久，小兰花也没法修复命簿。

小兰花叫苦连连，痛苦的喊着累。东方青苍无奈，只好答应她第二日再试试。东方青苍担心小兰花安危，不愿离开小兰花一步，毕竟如今他们两个的命连在一起。小兰花十分发愁，只好告诉他自己有喜欢的人。好不容易脱离东方青苍的视线，可一回到房间就看到东方青苍躺在床上。小兰花又气又急，可最后也无计可施。孤男寡女共处一室，小兰花怎么也无法入睡。东方青苍拿她没办法，只好乖乖离开房间。
