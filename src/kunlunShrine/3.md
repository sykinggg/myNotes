# 第3集：阿东贪心命丧轮回庙

胡八一仔细对照之后才发现原来顶棚的图和手中经书上的图合二为一才是完整的地图，并且通过图中提示找到了九层妖楼的位置，雪山代表的就是龙顶，守护的就是九层妖塔，雷显明佩服胡八一不愧是摸金校尉，王胖子故意逗雷显明，现在已经找到了位置，就不需要经书了，也可以让雷显明离开了，雷显明慌忙求饶，但也相信胡八一是一言九鼎的人不会耍赖。

晚上一行人找了山洞休息，王胖子编织了两条红色的毛线内裤，其中一条送给了胡八一，胡八一虽然嫌弃可还是收下了。

各自一帮人休息的时候，相互议论着对方的人，王胖子认为韩淑娜就是雷显明的情人，而韩淑娜那边也最讨厌倒酒是王胖子，整天扣没遮拦什么都敢说，雷显明只好安抚韩淑娜的情绪，毕竟现在要找到东西才是关键。

阿香却总是感觉到他们就在一个危险东西附近，显得心神不宁，雷显明虽然害怕，但也只能稳定大家情绪，催促赶紧休息。半夜王胖子醒来上厕所，却发现雷显明的助手鬼鬼祟祟出去，他慌忙将胡八一叫起来随后跟着。

却发现雷显明的助手阿东直奔白天去过的轮回庙而去，胡八一认为肯定是冲着银眼佛而去，两人一合计打算抄近道上去，将阿东给吓跑了。可二人刚进去庙里，就听见了打雷声，吓得外面的阿东惨叫声传来，但他依然大着胆子进来，胡八一和王胖子赶紧藏起来。

阿东进入到大殿里，先跪拜了如来佛祖，王胖子一泡尿实在憋不住了发出声音，听到声音的阿东吓坏了，四处查看，却什么也没有看到。大着胆子拆掉了那堵封好的墙壁，阿东进去搬动金佛的时候，却发现金佛上面拴着铁链子，他只好拽着铁链子一起搬出来。试图将铁链子拽断，铁链好不容易拉断了，可里面却出现了一扇门，门里面走出一个怪物，当场将阿东吓晕了过去，阿香此时也感受到了危险梦中惊醒。

一直趴在房梁上的胡八一和王胖子看着下面的怪兽，王胖子忍无可忍一泡尿洒了出来，气得胡八一责怪王胖子尿的不是时候，果不其然，怪兽顺着尿液发现了上面的人，胡八一让王胖子跳下去砸死怪兽，王胖子果然不辱使命，不偏不倚砸中了怪兽。

胡八一忽然看出怪兽曾经出现在壁画上，阿克说过是专门惩罚有业报的人，此时怪兽忽然醒过来袭击王胖子，胡八一跳下去和他一起对付怪兽，王胖子被怪兽压在身下，见阿东醒过来，胡八一叫着他一起对付怪兽，阿东生怕连累自己躺下装死。

胡八一和王胖子奋力将怪兽打飞，当怪兽子再次起来的时候，已经不见了胡八一和王胖子的身影，怪兽四处寻找，二人藏身在木板之下，阿东想要趁机溜走发出声响，怪兽直奔阿东而去，王胖子和胡八一听着阿东的惨叫声这才从木板下面出来，眼睁睁看着怪兽咬断了阿东的脖子。

王胖子和胡八一判断阿东已经没救了，就赶紧撤退，将怪兽关在门内，怪兽发出恐怖的叫声撞破了门，一嘴咬住了王胖子的衣服，胡八一只好让王胖子脱了衣服拉着他逃命，没想到怪兽再次扑过来，将胡八一扑倒在地。
