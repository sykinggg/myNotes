# 第5集：初一答应做向导

Shirley杨觉得这次去可能有希望治好红斑狼，因此询问胡八一治好之后的打算，有意要带着胡八一去她的家乡，言下之意一目了然，可胡八一却装糊涂，反而责怪Shirley杨太容易相信人，什么人都敢带回家乡去，再说他也听不懂那里的人说话，也没有朋友根本不打算去，Shirley杨知道这是胡八一变着法的拒绝她，生气之余也表示再也不会重提此事，一脚油门踩到底，吓得胡八一赶紧抓紧了扶手。

王胖子给大家分罐头，韩淑娜和雷显明都吃不惯想要去打猎，王胖子提醒两人这里是高原水都烧不开，两人无奈赶紧拿了罐头离开。王胖子故意给阿香送过去，想要和阿香套近乎，阿香故作害怕的样子，说王胖子背后有鬼，吓得王胖子不敢乱动，阿香趁机离开，王胖子还以为阿香是看到了他后辈的红斑狼，更是唏嘘不已。

当胡八一回来的时候，看着Shirley杨背着厚厚的行李包想要帮忙却被她拒绝，王胖子将找到初一的事情告诉胡八一，可是却无法说动初一给他们带路去喀拉米尔山。胡八一什么也没说，只是带着一个轮椅去附近找了格玛，他一直自责当初自己能力不强，不能很好保护格玛和大个，害得格玛成了瘫痪，终日躺在床上。

有了这个轮椅格玛就可以走出帐篷到外面看看，格玛很开心，让胡八一没有想到的是，格玛的哥哥正是初一，初一对胡八一救了格玛的事情很感激，也一直将他当做恩人，这次听说要去喀拉米尔山主动提出帮忙带路，他也是唯一去过那里的人，知道那里凶险万分。

扎西、格桑都是初一的好兄弟，也是 打狼的高手，这次跟着一起去，同时也给每人发了一杆枪，随时准备应急使用。当天晚上，大家都来到初一家的帐篷里做客，Shirley杨对胡八一心里有气，故意拉开了距离坐，初一告诉大家任何小车都开不进去，只能让牦牛托着行李去。初一也说起那里有一个怪兽比狼群还可怕，韩淑娜有些害怕，胡八一趁机提出让雷显明和韩淑娜以及阿香都留下，他可以保证将雷显明亲人的骨灰带下来，这次轮到韩淑娜表示反对了，赶紧解释雷显明是想要亲自迎了亲人回家，胡八一也就不再多说。

初一祭拜了亲人之后准备出发，女儿拉姆有些不舍，初一还特意给了拉姆一些糖，拉姆却表示要等着初一回来再吃。

初一带队出发，格玛也来送行，也表示每天都会为胡八一等人祈福，希望他们能早点回来。胡八一等人告别了亲人，也踏上了征程，越往上走空气越是稀薄，韩淑娜和雷显明很明显有些受不了，只能先留下，胡八一和Shirley杨跟着初一等人继续走。

接下来又来到一处天堑的独木桥之处，初一和胡八一先行过去，回头看着Shirley杨以为她不敢往前走。
