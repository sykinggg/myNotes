# 第28集：靖王被封 誉王起疑

&emsp;&emsp;静妃明白苏先生就是林殊。她郑重叮嘱靖王说苏先生是至诚之人，待他要比旁人更要亲厚几分，切莫忘了扶持的情分。她还特意做了双份的点心，命靖王带一份给苏先生。一连串的疑点让靖王确认《翔地记》有古怪，反复研究，却不得要领，令他备感困惑。

&emsp;&emsp;靖王被封亲王，加五珠冠，与誉王仅差两珠，朝政格局有了新改变，誉王因此十分气恼。梅长苏继续迷惑他说靖王加封是件好事，表示太子就快被废，皇上需要新的平衡局面。誉王烦燥的表示难道花十年时间斗倒了太子，自己什么也没有得到，还得继续与靖王斗？梅长苏安慰他说靖王无法与太子相提并论。誉王心事重重的离开苏宅。梅长苏心知这一次再也糊弄不住誉王，开始加强戒备。

&emsp;&emsp;四姐温柔体贴，成功的接近了童路，让他痴情深种，掉进了美人计之中。

&emsp;&emsp;靖王加封后，叮嘱母妃小心。这时，宫里传言四起，说靖王不输誉王，有些方面甚至强过誉王。皇后大为光火，誉王也开始怀疑梅长苏。

&emsp;&emsp;朝堂上，誉王与靖王已拉开战局。靖王学习政务，废寝忘食，进步很快。一日，沈追夜访靖王府，告知今年有五个州连遭旱灾和蝗灾，颗粒无收。以前，太子、誉王争当赈灾主事人，目的不是救百姓于水火，而是瓜分救济款。他俩每次赈灾后，灾区就会饿殍遍野，暴民四起，因为落到灾民手中的救济款不足三成。沈追恳请靖王为救灾民，争当这次赈灾的主事人。

&emsp;&emsp;翌日，誉王、靖王在梁帝面前为谁前去赈灾争论不休。誉王无奈，只得提出先捐出自己府中的银两安民，清廉的靖王囊中羞涩，无力与之竞争，赈灾一事落入誉王手中。

&emsp;&emsp;出宫后，沈追气愤的对靖王说，希望天上掉下一个雷，正好砸在誉王头上。靖王到苏府求计，叹息自己无能，颇为沮丧。言谈中提及沈追的话，梅长苏戏谑的说：有一个雷正在路上。
