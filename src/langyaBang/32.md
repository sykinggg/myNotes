# 第32集：劫夺惨败 引君入瓮

&emsp;&emsp;夏秋押解卫峥的囚车快到京城了。悬镜司严阵以待，要将劫囚车的来犯一网打尽。

&emsp;&emsp;梁帝回宫后怒斥皇后。皇后一口咬定自己不懂医术，担心龙体受损才会对静妃严加审问。静妃赶来求情，看透皇后心思的梁帝这才答应不予追究。

&emsp;&emsp;甄平、黎纲瞒着梅长苏，带着飞流及江左盟的一群高手埋伏在金陵西门处，准备劫夺卫峥。囚车临近，双方陷入激战。悬镜司高手云集，仓促行事的江左盟损失惨重，只得败退。甄平受伤撤走，幸得路过的豫津掩护才逃回苏宅。他们知道，卫峥关入悬镜司地牢后，易守难攻，施救将更加困难。

&emsp;&emsp;宸妃夜夜入梦，令梁帝疑神疑鬼。为求心安，他安排静妃替他暗中设立宸妃牌位，让她早入轮回，不要留恋阳间。

&emsp;&emsp;夏秋向夏江建议加强地牢防卫。夏江却说照常就行。因为卫峥的价值就是钓鱼，诱人来救。

&emsp;&emsp;药王谷的素谷主率众快马加鞭赶到金陵，与黎、甄等人商量营救卫峥的计划。醒来的梅长苏从飞流口中得知了此事，命令廊州旧部蛰伏，不得妄动。

&emsp;&emsp;靖王回京，列战英准备赶在靖王入宫前禀报卫峥一事，不料刚想开口就被誉王打断，随即便拉靖王入宫，列战英没有机会告诉靖王。

&emsp;&emsp;梁帝面前，誉王与夏江你唱我和，故意拿卫峥一事刺激靖王，咄咄逼人，步步相催。驳斥中，靖王不禁流露出对祁王和林帅的尊敬和怀念，那种孺慕之情重重戳在梁帝的心窝上，让梁帝勃然大怒。

&emsp;&emsp;靖王到芷罗宫看望静妃。准备离开时，一旁的小新故意说漏了嘴，为静妃被欺负一事打抱不平，还详细描述了梅长苏为行苦肉计，故意遣人拦住她和戚猛，不让他们送信搭救静妃。静妃察觉有异，连忙解释，靖王仍气愤不已。
