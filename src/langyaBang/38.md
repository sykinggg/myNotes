# 第38集：与狼周旋 命悬一线

&emsp;&emsp;梅长苏说卫峥早已逃出京城，夏江不信，他自信满满的认为京城布控严密，卫峥绝不可能出城。为掩护卫峥，迷惑夏江，梅长苏解释说卫峥藏在夏春夫人的箱笼中出了城，夏江这才明白了自己的疏漏之处，相信此言属实。

&emsp;&emsp;豫津来到纪王府，出示了卫峥的通缉令。在他的暗示下，纪王想起那日在小巷中看到夏冬转移的犯人正是卫峥。联想到靖王的艰难处境，他觉得应将自己亲眼所见告知梁帝。尾随纪王进宫的甄平连忙通知蒙挚作好准备。

&emsp;&emsp;梅长苏在夏江面前不卑不亢。他用独门手法折磨梅长苏，梅长苏疼出一身冷汗，仍面不改色。夏江质问梅长苏作为天下第一大帮的宗主为何要搅入朝局？为何要选择靖王，是否有自己的目的？梅长苏沉着应对，一一作答。夏江见他软硬不吃，只得拿出自己的独门毒药——乌金丸，威胁说如果不服解药，七天后就会毒发身亡，逼梅长苏说出实情。他进一步警告梅长苏如果梁帝提审时他敢乱言，自己能用隔空手法让他瞬时毙命，且毫无痕迹，到时梁帝会认为他畏罪服毒自杀。

&emsp;&emsp;纪王将亲身所见告诉梁帝后，梁帝深感事有蹊跷。他速将蒙挚召进宫中，命他直接去悬镜司带夏冬进宫，不得让夏江知道。

&emsp;&emsp;悬镜司内，梅长苏反守为攻。他诘问夏江，悬镜司一向不涉党争，为何此次要参与夺嫡，致靖王于死地？夏江百般狡辩，不能自圆其说。梅长苏轻蔑的对夏江说：你怕靖王，就像当年怕祁王一样，因为祁王曾力主裁撤悬镜司，所以你害死了祁王，害得七万赤焰军冤死梅岭。夏江被说中心事，恼羞成怒，质问梅长苏是否是祁王旧人？梅长苏回答道我是景仰祁王的人。夏江挥拳击倒梅长苏，强迫他吞下乌金丸。这一切都被悄悄潜在一旁的夏冬看在眼里，听在耳中。夏冬含泪跪下，求夏江拿出解药，及时收手，改过自新。夏江却充耳不闻，命人押走夏冬，严加看管。这时，蒙挚赶到悬镜司，不由分说将夏冬带走。
