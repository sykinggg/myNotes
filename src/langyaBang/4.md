# 第4集：靖王意动 霓凰遇险

&emsp;&emsp;梁帝无奈叫停比武。梅长苏当场献计，指出百里奇武功太过刚硬，只需几个孩子就可击败。梅长苏以退为进，激北燕当场应下五日之战。梁帝准梅长苏之提议，由蒙挚亲自在掖幽庭挑选了三个稚子出战，庭生是其中之一。

&emsp;&emsp;惠妃被罚去先太后佛堂点烛，靖王之母静嫔陪同前往。她俩无意听见一侍女向吴嬷嬷索要催情烈酒“情丝绕”----曾被先太后用在莅阳长公主身上，知无好事，甚感不安。

&emsp;&emsp;梅长苏告诉飞流速度是取胜关键，让其敦促三稚子加紧训练。霓凰来访雪庐，向梅长苏表明自己处境艰难，已无退路，请其力挽狂澜，并表明自己不知为何深信梅长苏。梅长苏让她放心，定会妥善解决。

&emsp;&emsp;静嫔在觐见太皇太后时，借香囊向莅阳长公主传书。莅阳公主应邀与静嫔相见。静嫔告之宫中有人要用“情丝绕”，两人不约而同想到受害之人将是霓凰。静嫔请求莅阳搭救霓凰。莅阳前往穆府送信，因霓凰、穆青外出而未得见。此事被谢玉得知。

&emsp;&emsp;三稚子进步颇快，受梅长苏称赞。靖王来雪庐看望庭生，讥讽梅长苏甘为幕僚，问他会选太子还是誉王。梅长苏趁机表明心迹，称愿选靖王，助其争夺皇位。靖王闻言哑然失笑，自嘲母族不显，人脉衰败，梁帝冷落，处境艰难，皇位定在誉王和太子之间，自己毫无希望。但是，虽然皇位远如浮云，若能斩太子和誉王至尊之路，靖王愿付任何代价。

&emsp;&emsp;深夜，蒙挚潜入雪庐，得知庭生是祁王遗腹子。梅长苏告知要扶助靖王，遭蒙挚反对。梅长苏心坚似铁，表示为雪沉冤，愿为此背负所有罪责。此时，莅阳突访，告诉梅长苏“情丝绕”一事，让他提醒霓凰小心防备。蒙挚在屏风后听闻此事，深感气愤。二人已知明日谁请霓凰进宫，即是幕后黑手。
