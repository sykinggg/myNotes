# 第47集：入主东宫 意外重逢

&emsp;&emsp;能让聂锋与夏冬早日相见，卫峥和梅长苏都觉得宫羽的计策可行，只待适当时机便可行动。

&emsp;&emsp;在蔺晨的引导下，飞流抓住了秦般若。甄平收到飞鸽传书，告知谢玉已死。梅长苏认为他的死可以利用，因为莅阳长公主手中有他一封亲笔供述，可以用来作为重提旧案的契机。

&emsp;&emsp;沈追看出靖王心思，担心他冒险犯上，得不偿失，下朝后主动与他谈及祁王旧案。他说13年前就有很多人对案件真相提出异议，可是非死即贬，再无人敢言。他直言不讳的指出，对梁帝而言真相并不重要，重要的是祁王羽翼太过丰满，犹如鼾睡在梁帝卧榻之侧，让他感到如履薄冰，岌岌可危。沈追叮嘱靖王，如今东宫未立，若冒然提出重审赤焰旧案，只会让梁帝认为他自恃新功，逼他承认当年犯下的大错而恼羞成怒，对昭雪旧案毫无益处。

&emsp;&emsp;蔺晨一到苏宅，梅长苏就急着让他为聂锋诊病。切完脉后蔺晨说聂锋中毒不深，其毒能解，梅长苏松了一口气。蔺晨把秦般弱当“礼物”送给了梅长苏。根据秦般弱的行踪，他俩推测夏江仍在京城。事实上夏江果然潜伏在外臣范大人府中，并计划利用范大人的侄女范贵人送一封密函给废太子的生母越贤妃。

&emsp;&emsp;梅长苏殚精竭虑，处处为靖王考虑，身体越来越差。蔺晨毫不客气警告他身体状况已不容乐观，如果再不静心修养，随时有生命危险。

&emsp;&emsp;靖王被册立为太子，入主东宫。梅长苏为不留后患，命人彻底封闭了密道。梁帝身体大不如从前，他终于意识到自己老了，遂令太子监国理政。

&emsp;&emsp;蒙挚安排人手换出了夏冬。得知夏冬要来，聂锋既满心期待又惴惴不安，怕夏冬接受不了他的容貌。看着面目全非掩面垂泪的聂锋，夏冬深情的握住他的手，千言万语只化为一句话：“你终于回来了！”阴阳两隔又骤然相逢，二人已分不清是喜是悲，默默两无言，唯有泪千垂。

&emsp;&emsp;蔺晨将火寒毒症状描述给众人，并列出两种解毒方案。一种是搓骨削皮，碎骨重塑，解毒后面容虽改但与常人无异，只是内力全失，体弱多病，如果保养得当最多能活到四十岁；而另一种解毒方法痛苦少且可享天年，但口不能言，体外白毛也不能尽除。

&emsp;&emsp;在场的蒙挚、霓凰等人这才知道中了火寒毒的林殊经历了怎样的锥心之痛。霓凰的心都在颤抖，蒙挚更是不管不顾的吼叫起来，斥责梅长苏不该不顾惜身体，来到京城以命相搏。

&emsp;&emsp;为长相厮守，聂锋和夏冬选择了第二种解毒方案。
