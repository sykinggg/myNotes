# 第52集：滔天罪行 惊世骇俗

莅阳走后，景琰安慰失望伤心的梅长苏，准备让言侯来做殿前首告。

景睿明白母亲的矛盾和酸楚。一旦翻案，势必株连谢弼和谢氏族人。莅阳长公主本想用出面首告求一个恩赦，话没说清楚，就被太子生气的逐了出来。

莅阳长公主认为太子雪冤之心已坚，到时谢府必受株连，而且自己如果袖手旁观，下半生将寝食难安。她下定决心，去而复返，答应了景琰的请求。景琰主动保证赦免谢氏满门株连之罪，长公主这才知道不是真心助他的人，他也不愿引以为援。梅长苏知道这是一场硬仗，提醒景琰要考虑变数，备好后手，而景琰早已联络群臣，切断了梁帝与外界的联系。看到景琰渐趋老练的处事手段，梅长苏感到欣慰。

梅长苏请景琰在梁帝寿宴当天带自己同去，景琰激动的说那是理所当然，一定会让他去见证最后的结果。梅长苏的恭谨守礼让景琰难受，他让梅长苏在他面前就是林殊。梅长苏一声“景琰”唤回了二人美好的记忆。相谈中，梅长苏露出冤案昭雪即会隐退之意，令景琰十分不舍。

看到院中学习射箭的庭生，梅长苏建议景琰保守庭生身世秘密，因为没有出生玉碟，庭生不可能恢复皇室身份，景琰决定将庭生收为义子，让他无忧无虑的生活。

梁帝寿诞之日来临，景睿鼓励长公主沉着应战。景琰携王妃与梁帝共进早膳。寿宴开始后，群臣轮流贺寿，梁帝笑逐颜开。这时，莅阳长公主却只身前来跪拜，表示自己不只是前来贺寿，还要代罪臣谢玉供呈欺君罔上、陷害忠良的大逆之罪。她高举手书，不顾梁帝反对，一一细数谢玉罪状。梁帝连连喝止，长公主却置若罔闻，继续揭发案情真相。梁帝龙颜大怒，命御林军将她带下，但没人听命。这时，蒙挚跪求梁帝让长公主把话说完。莅阳趁机将谢玉五大罪状尽数列出。她清朗的声音响彻大殿，谢玉之罪霍霍滔天，令人惊骇。宗亲和群臣们再也坐不下去，纷纷出来要求重审赤焰旧案。望着跪了一地的朝臣宗亲，梁帝颤抖着声音问：你们要一起逼朕吗？
