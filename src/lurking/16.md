# 潜伏第16集

&emsp;&emsp;翠平急切的期望她与余则成的关系能有进一步的发展，不懂得恋爱技巧的她尝试着向余则成示好却总是不得要领，经常弄得二人尴尬不已。九十四军内部有人私自倒卖军需物资，李涯奉命负责抓捕工作。而负责情报工作的陆桥山为了排挤李涯并没有把准确的信息全盘告之，使本想好好表现一下的李涯错失任务，更加狼狈。吴站长也不想理会太多政事，只想多敛些钱财，以备退位时用。正值九十四军想息事宁人，愿意出大价钱来遮丑。余则成借此机会帮站长大捞了一笔，讨得了站长的欢心，在陆、李相争的夹缝中，受到了重用。为了投桃报李，站长有意提携余则成为中校，好在日后有机会晋升副站长一职。余则成让翠平向站长太太讨教黑市买卖的窍门，翠平误以为他想敛财而非常鄙夷。而得知余则成是想将这些不义之财上交给组织用来装备我方军队的时候，她立时改变了态度，一头扎进了官太太聚集的黑市中，当然，她对余则成的爱意也越加深厚。
