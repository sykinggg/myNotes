# 潜伏第23集

&emsp;&emsp;陆桥山回到天津后，作威作福，大肆镇压学生，很多人命绝在他的枪口下，天津的地下组织损失惨重，余则成真后悔当时一时心软，只想马上除掉这个祸患。站长和李涯也怕陆桥山反攻倒算，打算主动出击，枪杀学生代表的负责人，嫁祸给陆桥山。陆桥山一时被推到了风口浪尖。从“假夫妻”变成“真爱人”，余则成和翠平都觉得有必要向上级汇报，余则成事事以工作为先却让翠平误会为他不想向上级汇报他们的关系的转变。单纯的翠平毫无恋爱经验，心直口快的她经常会把柔情蜜意的气氛变得尴尬不已，余则成对此十分无奈。站长和李涯的嫁祸不但没有让陆桥山害怕，反而让他变本加厉。为了除掉大肆残害进步分子的陆桥山，余则成和罗掌柜制定了一个周密的暗杀计划，但由于负责狙击的同志来不及赶到天津，狙击陆桥山的任务落到了翠平身上，翠平不辱使命，拿出了打游击时练就的百步穿杨的枪法，不仅干脆利落的解决了陆桥山，还制造出枪击余则成的假象使他不受牵连。
