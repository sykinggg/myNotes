# 潜伏第25集

&emsp;&emsp;翠平将许宝凤安置在一家客栈,许宝凤用谢若林提供的情况作诱饵,翠平很快就向她暴露了真实身份。翠平哪里知道，她和眼前这位“好同志”许宝凤的对话，都被事先藏在皮箱中的微型录音机录了下来。谢若林将录音带卖给了李涯，本来已经心灰意冷的李涯又开始蠢蠢欲动。本以为帮助了战友的翠平，把一切告诉了余则成，余则成断定翠平中了圈套，好在事情还有挽回的余地。为了控制局面，安抚慌乱的翠平，余则成并未将实情告诉翠平，而当下之急，就是要先转移罗掌柜的交通站。新的交通站迅速的建立起来了，联络人是执法队的廖三民。余则成借廖三民之手监禁了许宝凤，本以为此事很快就会风平浪静。不料尚不知情的翠平救许宝凤心切居然私自前往已经转移的罗掌柜的交通站，狡猾的李涯略施小计，翠平彻底暴露了！余则成对翠平进行了严厉的批评，翠平也终于明白了许宝凤的真实身份并认识到问题的严重性。经过仔细分析，余则成觉得还有机会翻盘，于是他决定将计就计。
