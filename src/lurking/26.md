# 潜伏第26集

&emsp;&emsp;站长听了李涯提供的录音带之后下令逮捕余则成夫妇。李涯本以为这次绝对稳操胜券，不料余则成早已准备好了应对措施，他事先教了翠平一套说辞，让她在面对李涯的审问时滴水不漏。随后他又拿出拼改过的李涯和许宝凤的对话的录音带，把李涯的录音证据全盘颠覆，又把早已去见阎王的谢若林嫁祸为始作俑者。最后他还反客为主，以不学习先进的间谍知识为名狠狠的批评了李涯。面对余则成的机智和果敢，李涯的阴谋再次落空，余则成将局势紧紧的控制在自己手中。经过这次风波，余则成与翠平的感情也更上一层楼。秋去冬来，余翠二人相处得越来越融洽，此时又传来了我军攻占东北的消息，战事很快会波及到天津，站长吴敬中已经无心理会政事，他把最新的秘密潜伏计划----“黄雀行动”交给好强的李涯全权处理，自己却忙于将财产转移到台湾，并企图说服已被他视作心腹的余则成跟他一同离开。余则成当然不会跟他走，他和廖三民正忙于打探天津的城防部署，以便为我军攻打天津献上一份大礼！
