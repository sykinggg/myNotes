# 潜伏第28集

有一批用于“黄雀行动”的专配物资运到了天津，由李涯专管。余则成本想让廖三民打着官方的旗号去探听虚实，但李涯态度强硬让廖碰了钉子。李涯无意中发现了一直被廖三民关押的许宝凤的下落，许宝凤向李涯解释了余则成赖以翻身的录音带的来龙去脉，李涯恍然大悟并说服她为自己工作。廖三民汇报了李涯强行带走许宝凤的情况，余则成便知道当初偷梁换柱的计策再也无法瞒天过海，但是好在自己目前尚未暴露，为今之计，只能先送翠平出城。新婚燕尔的余则成翠平夫妇，虽依依不舍，但是为了完成潜伏任务，也只能劳燕分飞。殊不知这一切都已在李涯的监控之中，他将这些情况汇报了站长，并派许宝凤与几名特务一路跟踪翠平出了城，只待辨明接应翠平的人是共产党，便可以将余则成和廖三民一举拿下。许宝凤带领李涯的特务将翠平控制在郊外的一所民宅，负责接应翠平的同志赶到后双方经历了一场混战，这些同志都光荣牺牲了。无奈之下，翠平拉响了防身用的美式攻坚手雷。
