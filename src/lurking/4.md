# 潜伏第4集

&emsp;&emsp;组织上派给余则成的冒牌夫人终于有了眉目，而习惯了单枪匹马的余则成却只希望这个假夫人证实了自己的已婚身份后便马上离开。余则成受命加紧敲诈穆连成并结识了穆连成的侄女，一个新派的女学生----晚秋。同时余则成也发现当年刺杀吕宗方的马奎也在军统天津站任职，但他很怀疑余则成的真实身份，并开始暗中调查。面对多方压力的余则成四面受敌，而此刻秋掌柜又偏偏传来了组织上派给他的冒牌夫人出了问题的消息。余则成焦急万分，为了彻底消除敌人对余则成的怀疑，组织上决定临阵换人，另派一位假夫人到天津与余则成生活在一起并协助余则成工作。穆连成为了保命，愿意拿出酒厂来贿赂吴敬中，正好其侄女晚秋心仪余则成，穆连成更希望通过与军统联姻，来保住自己的家产，吴敬中似乎也愿意促成此事以获得更多的利益。而此时组织上派给余则成的“太太”终于到了，一见到这个叫翠平的女人，余则成不禁暗中叫苦。
