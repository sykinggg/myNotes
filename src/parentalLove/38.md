# 第38集：江德福跳舞被嘲笑

九十年代来临，江德福与安杰已是满头白发，一天安杰抱着孙女在屋中四处寻找体温计，遍寻无果之下只得向江德福求助，江德福找到了体温计递给安杰，安杰拿过体温计替孙女测量了一下，发现孙女并没有发烧。

广场上，江德华与葛老师等人跳老年舞，正当所有老年人沉浸在跳舞带来的乐趣中，江亚菲忽然出现将音乐关小，来到老年人身边宣布八一建军节军区举行小组跳舞比赛。

回到家中之后，江亚菲将跳舞的事情告诉给了安杰，安杰听完江亚菲的话立即决定参加跳舞比赛，由于找不到合适的人选，安杰只得让江德福做舞伴，江德福已经多年不跳舞，再加上人老手脚笨拙，初次与安杰在房中练舞的时候频频踩中安杰的脚背。

虽然不懂得跳舞，但江德福依然信心十足决定陪着安杰参加跳舞比赛，由于孙女还要需要人照顾，江德福打了一个电话给老丁，将军区举行跳舞比赛的事情说了一遍，老丁听完之后不以为然，并没有为江德福感到高兴。

江德福在电话中提出让江德华照顾孙女，老丁一口回绝了江德福的要求，谎称江德华扭伤腰部无力照顾孙女。

江亚菲得知姑姑江德华扭伤腰部，赶紧来老丁家中探视，结果发现江德华行动自如腰部根本没有受伤。

江德福穿上白色西服与安杰来到广场做最后的练习，为即将到来的跳舞比赛做好准备，江亚菲来到操场上看着江德福笨拙的舞势，把心一横找来二个同事打算淘汰江德福，二个同事猜到了江亚菲的用意，不顾江亚菲哀求逃离广场，江亚菲无奈之下只得来到江德福身边，语气婉转指出江德福舞姿过于笨拙，江德福听出了江亚菲话中之意，没好气地询问江亚菲是否打算开除他，江亚菲见江德福猜出了她的心思，只得硬起头皮开除了江德福。

江德福当众被开除怒气冲冲回到家中生闷气，江亚菲赶紧买了一盒印泥回家，老丁见江亚菲买了印泥立即拿到手中观瞧，江亚菲无意与老丁搭讪，径直回到家中哄劝江德福，由于江德福依然在生气，江亚菲只得来到老丁家中求助。

老丁对印泥爱不释手，要求江亚菲再刻两只印章，江亚菲为了让老丁帮忙劝慰江德福，只得同意了老丁的要求。
