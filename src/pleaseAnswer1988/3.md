# 第3集：德善的懵懂初恋时光

&emsp;&emsp;德善的女同学们告诉她，善宇显然喜欢着她。迟钝的德善这才反应过来，她面对善宇的时候开始不好意思，为了跟善宇吃个拉面，还大晚上地化个浓妆，弄得家人和善宇都很诧异，德善的初恋开始了。

&emsp;&emsp;为了得到奖品，德善要参加才艺表演，她向得过舞蹈一等奖的东龙请教，大家一起在他们的老聚会地点——阿泽家，看她跳舞。东龙教得卖力，可德善却怎么也跳不好，弄得善宇和金正焕直想笑。阿泽听说善宇和德善的中学都有到庆州的修学旅行，很羡慕。不过大家更羡慕他去年赚到了一个亿，德善惊得嘴巴都快掉了。

&emsp;&emsp;父母把相机交到德善手里时，千叮咛万嘱咐，可丢三拉四的德善还是大意地弄丢了相机，发现后她吓得大哭起来，老师因此专门向她母亲打电话求情，这才使她放下心来。一起表演舞蹈的好友曼玉和祖贤摔伤了，德善为赢得比赛，想找善宇帮忙，却被正焕看见，逼着她替他藏酒。这时，东龙叫住了她。德善利用东龙好色的特点，说王子贤长得像王祖贤，张美玉像张曼玉，好容易才说服东龙帮忙。

&emsp;&emsp;当晚，善宇东龙正焕冒险违背校规，果然如约而至，他们的舞蹈消防车成为整个才艺表演的亮点，德善如愿得到了奖品。听到东龙说要见美女，德善叫出了祖贤曼玉，两人的尊容有点惊悚，吓得东龙连忙找借口开溜。

&emsp;&emsp;三人走出没多远，又跑了回来，原来教导主任——东龙父亲发现了他们，追了过来。逃跑的过程中，正焕和德善躲到了一处，狭小的空间里，累了的德善将脑袋靠到了正焕的胸前，正焕渐渐地有了反应。

&emsp;&emsp;回到家的德善，遭到了母亲的责打。躲避追打的德善，看见上学的正焕经过，急忙藏到他的身后，搂住正焕的腰，正焕有点不知所措的时候，德善妈一下子敲到了他的头上，德善妈呆住了，她和德善一起看着也呆住了的正焕。
