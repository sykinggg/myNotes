# 第23集：蔡元培遭训话质询 北大发起绝地反击

众人正在讨论，一人气喘吁吁地冲进来，说是门外有一群警察正在找寻易群先。易群先不愿回家请求同学们帮忙，延年冷静指挥，让人带易群先跳窗出去到法文进修馆，女生回宿舍掩护，男生随他去开门。警察搜查无果只好灰溜溜的离开，易群先来到法文进修馆翻墙而入，总算躲了过去。

次日，同学们一起找着那个记者踩人，看着柳眉的指认郭心刚这才发现这记者就是张丰载。众人见机行事拦住张丰载，柳眉佯装和易群先吵架，把张丰载约出来说着自己对陈延年已深恶痛绝，必要将陈独秀父子的丑事全部爆料出来。张丰载听到这里以为柳眉已同互助社离心，颇为自豪的将他们联合易夔龙，找记者写诋毁陈独秀文章的事情一一说出，恍然大悟地柳眉内心充满了愤怒。

钱玄同又将一大堆的稿子送来新青年的编辑部，众编辑不停的审核源源不断送来的文章，可是忙的不可开交。这边教育总长傅增湘又将蔡元培请了过去，他虽然支持新文化运动但现在各方压力实在太大，但蔡元培认为这是原则和公理，不是能够妥协的事情。张长礼拿着北大的杂志到处告状，要弹劾傅增湘和蔡元培。桐城派的诸多学者在林纾的带领下纷纷将矛头对准了蔡元培，吴炳湘发来的监控名单中也写着蔡元培和陈独秀的名字，蔡公现在是腹背受敌。但蔡元培淡然起身，对傅增湘说的这些他都已清楚，他表示自己不是蛮牛之人，能通融的地方他定会通融，随即告辞离开。

回去的路上蔡元培心中苦闷，下了车随意买了一份《公言报》，却看到了林纾写下的致于他的公开信。林纾将诸多同仁叫来家中，对自己写给蔡元培的这封信颇为得意的进行讲解。红楼内李大钊听着对工读互助会不实报道的事情，商讨着解决办法。延年决定先说服易夔龙，并找到张丰载恶意诋毁的证据再行反击。易群先表示章士钊曾是父亲易夔龙的上司，他若能出面便能说服成功。这时，蔡元培派人来将李大钊叫走，很是紧急。

蔡元培将林纾在报纸上发表的小说《妖梦》拿给他们看，鲁迅将小说中的人物故事一一解读，这明摆着就是讽刺蔡元培和他们这些新文化的倡导者是臭不可闻的粪土。安福系政客也剑指北大，对陈独秀、钱玄同等人进行点名道姓的指责，并要求将他们革职。树欲静而风不止，蔡元培已经如此忍让他们仍是欺人太甚，蔡公决心披坚执锐，上阵与他们展开正面厮杀。

鲁迅在北大礼堂内对林纾写的小说进行公开的批判，引得学生们纷纷鼓掌叫好。李大钊走上街头，对着群众进行公开演讲，痛斥旧势力的虚伪，高呼新文化运动万岁。毛泽东也登上演讲台，他呼吁着群众们要追求自由和解放，他坚信总有一天中华民族的改革会比任何民族都彻底，中国定会迎来光辉的一天。

林纾对张丰载没有追回《妖梦》一文的行径深恶痛绝，他大骂张丰载就是一个阳奉阴违、道貌岸然的小人，张丰载装模作样地跪倒在地，张长礼却劝说撕破脸也好，反正下周他们就要在国会上公堂对质。林纾终于明白文人和政客道不同不相为谋，他轰走张长礼，哭着说道再也不会与他们相见。

互助会的学生召开会议安排解决负面报道的问题，他们兵分三路，分别从劝解易夔龙、揭露和找寻日本记者报道张丰载丑行以及收集整理证据材料三方面进行反击。张丰载向张长礼说着自己内心的不安，他写给蔡元培的信是有去无回，他就期待着蔡元培发火好让他的那套言论不攻自破，却没能得逞。张长礼让张丰载毕业后去警察厅工作，但张丰载竟仍旧想要留在北大。张长礼毫不客气地说他是假道学，流氓身上披着个文人的外衣，这可让张丰载听着不高兴，自己现在这样还不全是他教的。

张长礼后天就会去国会进行提案的辩论，他打着如意算盘想拿易夔龙当枪使，自己躲在后面做壁上观。但他也深知想要借此将蔡元培拉下台根本不可能，他搞这么一出闹剧就是为了给蔡元培等人一个下马威，并给民国政府施加压力。同时，他安排张丰载一定要召集大批记者，只有通过舆论他们才能造起更大的声势。
