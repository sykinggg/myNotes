# 第3集：陈独秀为筹钱竟然卖字 李大钊被开除决心革命

陈独秀在书法上盖上章，还称要收润笔费，一幅字两百大洋，还要每月写一幅。一旁的汪孟邹哎呦哎呦的喊着帮腔，陈子寿陈子沛一下就明白这是醉翁之意不在酒。但陈独秀墨宝难得，他们也就笑着欣然接受。

汪孟邹拿着西瓜给在家里帮工的延年乔年送来，看着桌上编起的蝈蝈笼子他再三确认这回他们兄弟二人不会再闹事后，才走出去同陈独秀等人会面。章士钊回国后也赶来同大家座谈，陈独秀宣扬的反袁思想已深入人心，但袁党反扑力度也大增，各地都牺牲了很多人。章士钊传达孙中山的指示，袁世凯对反袁人事的大力镇压，但高压之下反而让革命党获得了大量的群众支持。孙中山和黄兴要求他们极力行动起来，一旦时机成熟立刻举兵护国，再造一个崭新的共和国。邹永成以回乡办杂志为名，阻止了一支两千多人的队伍，如果起势他必然追随。陈独秀同易白沙办起《青年杂志》，意在鼓吹新文化，造就新青年，用科学与民主的旗帜重塑一代青年。杂志封面上印刷的外国人正是卡内基，他想要利用卡内基自强不息的成功经历激励当代青年靠自己的努力来改变命运。

上海震旦学院礼堂内陈独秀在一片热烈的掌声中发表演讲，他向坐在台下的学生解释着自己创刊的用意，他分享着自己这些年的觉悟，所有的觉悟最根本的是思想道德理念的觉悟，人的觉悟。要找到这世界上最先进的理念来塑造国人，塑造国家。当代中国青年应有6点标准，一自主的而非奴隶的，二进步的而非保守的，三进取的而非退隐的，四世界的而非锁国的，五实利的而非虚文的，六科学的而非想象的。总得来说就是科学与民主并重，他们是一言一行的准绳，如有违反两者不论是圣人的言论祖宗的章法都是不足为信的。中国要强大就要挖掉腐朽地脓疮，青年要从说话写字做起，说白话文写大家都能读得懂的文章，以此来武装起自己推动中国的前进。

1915年9月15日，陈独秀创办《青年杂志》，现在思想启蒙的新文化运动如火如荼的开展。李大钊得知陈独秀创刊事情，他拿着《青年杂志》激动万分，两个月前他还想着要和陈独秀辩驳一番，现下又成了他的死党。倒袁已成为全中国的共识，唤醒民众的思想觉悟是李大钊一直想做的事，他已经决心放弃学业，并已收到陈独秀的来信。

湖南长沙的街上正下着绵绵细雨，士兵骑着高头大马挥舞着鞭子耀武扬威，一名身穿长衫的男子布包正慌张地跑着，他是第一师范学校的学生，名叫毛泽东。他看着坐在车上吃着三明治的富家少爷和街边灰头土脸正被人贩子推销的少女，还有地上匍匐的乞丐老人翻着垃圾吃，却也只能急匆匆的离开。

陈昌念着什么二十八画生的征友启示，何叔衡给大家分着洋柿子，蔡和森的母亲葛健豪倒觉得这个学生起势不小，他的本意是想要找到志同道合的人一同报效国家。这二十八画生就是毛泽东，萧子升，蔡和森，此时有人撑着伞而来说着要找二十八画生应征会友。毛泽东急急忙忙的赶回来，大声叫着自己淘到宝贝了，转头看见的这位新朋友名叫李立三。几个人就此坐下分享着毛泽东口中的宝贝，原来就是陈独秀主编的《青年杂志》。

买了一堆书的毛泽东甚至花光了他的饭钱，但他称有此书足以。几个青年人坐在一起，一边吃饭一边谈论着新青年的六条标准，毛泽东觉得还应该加上一条健壮的而非体弱的。他认为当今中国文弱者多，武风不振，只有他们强健体魄才能敢于同列强搏斗。他要投稿《新青年》杂志，好好谈一谈中国人自己的体育。毛泽东被《新青年》打开眼界，他在日记里写下这样一句话，与天奋斗其乐无穷，与地奋斗其乐无穷，与人奋斗其乐无穷。

天津南开中学小礼堂内，周恩来手握《新青年》杂志大赞陈独秀写的创刊词吸引人心，就像一盏明灯照亮了他的心灵。他声情并茂地朗读着杂志中的文章，将科学与民主的思想传播给坐在面前的同学们。街上唱着板的艺人都在唱诵着德先生赛先生，《新青年》成了大热的杂志。胡适特意写信发予汪孟邹，陈独秀读后激动此人是后生可畏，必须要让他这样的人才来到《新青年》。汪孟邹说着胡适是个孝子，学成后定会回国，陈独秀坚定地说道只要他回来就一定要抓住他。

高君曼的肺痨越发厉害，咳嗽地坐在板凳上缓着气。陈独秀想要去买药，但家里的钱已经花光，他只好去找汪孟邹。汪孟邹正在算账，《新青年》卖出两千本本金已经回笼，陈独秀开口让他也算算自己的稿费和编辑费。汪孟邹算着他实际应得的一百二十块大洋，还提起了要陈独秀拉一些富商入股。

延年乔年正好回来，他们又要在搬运站打工，又要在汪孟邹这里干编务，还不要工钱说是抵了住宿费，晚上还要学习法语实在辛苦。听闻此话，陈独秀竟然提出把二人的编务拿给自己。这话可让乔年不满，他跑出去走到陈独秀面前，陈独秀连同自己的稿费一同给了儿子想着就交给他俩算学费。延年将钱还给陈独秀并同他对峙，他不能既想爱国又想爱家，他这样的人只会给家人带来伤害。陈独秀恼火儿子这般坚持只会让自己受苦，但延年认为自己坚持的是正确的方法。话已至此，父子二人纷纷恼怒离去。

李大钊被早稻田大学以长期欠习为名除名，可他却反而松了口气，终于可以心无旁骛的搞革命了。他热爱编辑杂志，国内如今的形势他必须要回去，他决定将《民彝》杂志在海外办好后就回国。《新青年》迫于压力停刊，李大钊却在为《新青年》的复刊号写文章，这种小阻碍无非就是换个名字而已。他读着写下的名为《青春》的文章，他以昂扬的精神和文字表达出了唯物主义思想，提出再造青春中华的理想主张，号召青年不断改造自我，建立青春之中华。

一品香旅阁内，吴稚晖笑着拉着延年乔年进屋烤火，还抱怨着陈独秀这回国了都不知过来看看自己。《新青年》被人状告侵犯名誉权而停刊，陈独秀只好忙着打官司，这状告人简直是自取其辱。吴稚晖此番回国，一为倒袁运动，二为启动留法行动。得知两兄弟想要去法国留学，他询问二人当今西方最流行的理论是什么，乔年回答是进化论。陈独秀在学校里发表过演讲，他们兄弟二人也很受启发。吴稚晖却告诉他们，当今最流行的理论是克鲁泡特金的互助论，他还特意从法国带回理论资料交给两兄弟。

1916年3月，袁世凯被迫取消帝制，一场丑态百出的闹剧终于夭折。随后李大钊应邀赴北京担任《晨钟报》主编，此时的他已经成为了充满新思想的革命家。北京前门大街上李大钊坐着人力车询问着车夫的生活状况，车夫直言他们现在的日子是苦不堪言，就没过过这样憋屈的日子。
