# 第33集：学生被强制看押学校竟成监狱 陈家救国之心不灭父子终和解

此次游行工人们也纷纷加入其中，李大钊得知被关押的学生们因为被暴力逮捕而受伤，加上没水没药很多人都生了病。李大钊想要去探望，但他自己已被盯上实在不便前往。李大钊提出样大家团结一致，对被看押的学生要有钱捐钱有物捐物，每个人都做出自己的努力。

百姓们纷纷堵在路口要求送水送食物，吴炳湘高兴老百姓可是帮了他的大忙，欣然应允百姓们给学生送食物和衣物。柳眉和君曼带着食物来看易群先等学生，学生们终于有了吃食和短暂的与亲友见面的机会。赵世炎来到陈独秀家中，将游行学生与军警展开斗争受伤，以及胡仁源要正式接任北大校长一职的消息告诉他们。现在，全国上下学生罢课，工人罢工，商人罢市，所有人都参与到了爱国运动中，他们期盼的胜利就要不远了。

赵世炎发现了刘一品在盯着李大钊，他们轮番替换守在李大钊的身边让他决不能随意出门。李大钊在北大校园里带领学生们制作横幅和海报，安排着学生运动的情况。怀孕的赵纫兰带着食物和换洗的衣物来到校园内看他，李大钊对妻子的体贴是又感激又懊恼。学生们散去，疲惫的李大钊终于坐下来吃着赵纫兰做的饼和汤，赵纫兰看着北大校园她自小就羡慕李大钊可以上学，但作为旧中国的女性她根本没有机会走入学堂接受教育。李大钊提出带着她去校内转转，他摸着妻子的肚子很是愧疚，自己一直忙于学生运动都无暇顾及家人。但赵纫兰对他很是理解，更是提出过些天带着孩子回老家，在家里生孩子不让他操心，并将蔡元培扣下的薪水交给他，让他拿去做自己想要也该做的事情，有妻如此也是人生一件幸事。

陈独秀提着公文包想要出去走走，但家人们拦着他不让他出去，可他陈独秀真想干什么是谁也拦不住的。学生们高喊着严惩卖国贼的口号在街上奔跑，因为身后就是一大批带着枪骑着马的军队士兵。宪兵队在学校门口架起机关枪看管着学生，百姓们沿街做着食物就是为了能及时给被看押的学生送饭。学生们抓着窗户上的栅栏仍旧高呼拒绝巴黎和约，维护国家主权。看到此情此景，陈独秀的心中无比动容他独自走入空寂的北大办公楼内，靠着楼道的墙壁摊坐在地，哼着故乡的小曲，陈独秀回想着这些年的一幕幕事件和一个个人。

胡适和李大钊着急地跑来寻找着陈独秀，陈独秀听着亲友的担心寻找却是不愿出声。李大钊和胡适在走廊里踱步希望能等到陈独秀，李大钊激动地向胡适询问，现在这样还是值得他们为之奋斗的民国吗？本该读书育人的学校竟成了爱国学生的监狱，他不明白这个国家到底是怎么了。胡适认为如果当初不采取总罢课，采取缓和的方式就不至于到现在的状况，但李大钊坚决认为应该同反动政府斗争到底。一向儒雅沉稳的胡适终于爆发，他担心北大、担心学生，也担心陈独秀和李大钊会毁了自己的学术前程和美满家庭，他更怕新文化运动就此被彻底摧毁。李大钊扶着几近崩溃地胡适，他坚信一个国家若是强权压倒了公理，那人民就有行使民主的权利，中国需要一场彻底的革命，要把思想和行动觉悟结合起来，为此他九死而不悔，虽千万人亦往矣。

陈独秀默默听着他们的讨论，起身站起来只说要回家好好待着，吃饭喝水。胡适流着泪喊住他，陈独秀停住又回头告诉他们共和死了，但他不会死。

家里，众人都在等着陈独秀回来吃火锅，他进了门却是倒了杯酒端起来，向妻子和孩子道歉，他既不是合格的父亲也不是合格的丈夫，也不是一个好老师。他是一个立志救国的人，他不能眼睁睁看着学生们受辱却无动于衷，他必须为这个国家做些什么，因此家人们很可能还会因为他而吃苦受罪，为此他非常的愧疚和无奈。延年和乔年也向父亲致歉，他们已立志为国献身可能无法为父母尽孝为亲友尽责，延年喊着“爸”为陈独秀斟满酒杯，这么多年过去，父子俩的心结总算在今日得以解开。
