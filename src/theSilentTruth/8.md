# 第8集：黄毛翻供致使朱伟刑拘

夜幕降临，江阳开车载着朱伟前往苗高乡，彼此聊着生活与经历，尤其朱伟每次谈及儿子，脸上总会挂着得意笑容，仿佛若干年后看见父子俩联手抓罪犯的场面。交谈期间，途径鱼塘听到救命呼喊，待二人循声追去，才发现黄毛遭人绑架，眼见朱伟对空明枪，立刻撒手把黄毛抛在车外，调头逃离。

黄毛得以获救，再三考虑下，便将事件原委逐一道出，承认当年曾为卡恩集团物色年轻女性，因被侯贵平意外发现，并且掌握学生遭受性侵名单和照片，所以才会惹来杀身之祸。朱伟拿着江阳的录音机给黄毛录音，决定带他回警局继续审讯，胡一浪欲杀黄毛未果，索性吩咐李建国故技重施，再以审讯程序违法为由，带着律师赶到警局打断审讯。

朱伟怒不可遏，与李建国发生冲突，幸好其他警员及时阻止场面恶化。李建国直接向督察处举报，李建国临走前将录音机偷偷交给江阳，叮嘱他务必保住黄毛这条线索。

由于孙传福投诉朱伟非法传唤，再加上黄毛临时翻供，嫁祸朱伟开枪威胁他诬陷孙传福是黑恶势力。面对当前种种变故，江阳身为检察官却深感无力，只能眼睁睁看着朱伟揽下所有责任，等待司法审判。陈明章建议江阳切莫冲动，不要辜负朱伟的信任与期望。

江阳亲自去感谢吴检出面说情，同时希望吴检能把朱伟弄出来，然而吴检是想让他放弃调查，做好本职工作，以免惹祸上身。如果换做从前，或许江阳定会稍作考虑，可是如今经历这么多事，他却再也无法选择沉默，一旦开始就不能停止，哪怕是死亡。

最高检在全国遴选年轻有为的检察官，吴爱可正有此意，吴检劝不动江阳，便让他尽量远离女儿。恰好江阳不想连累吴爱可，毅然提及分手，他不待吴爱可回应，转身离去，默默爬上长阶，一步步迈进黑暗。

起初检察院碍于大人物要求，只能先对朱伟下达刑拘单，但因多名政法界人士站出来发声，要求从轻处理，甚至写信给各级领导，包括人民群众的联名信提交，最终决定对他取保候审，暂免一切职务，送去警官学校进修三年。

严良主动找张超谈心，得知张超父母早在2008年年底去世，而老年扶手则在2009年6月批发生产，所以时间点成为关键谜团。对于侯贵平死因，严良十分好奇张超为何没有替他主持公道，但此时张超已和李静订婚，离开政法学院，甚至告诫江阳不要再把李静卷入案件当中。至此，江阳仅剩独自一人。

张超母亲去世后，房子便租给江阳，由于江阳生前没有骨折记录，所以老年扶手应该还有额外作用，比如扶手易卸，与伸缩衣架互相替换。严良据此猜测凶手另有其人，或者江阳蓄意自杀，法医在案发现场尚未找到任何微物证线索，即便死者与凶手事先策划好，可是人的求生本能仍会强制启动，从而进行挣扎，但是江阳的尸检报告里发现他在死前处于放松状态。

李静将侯贵平亲笔信件交给江阳，里面包含一张金山照相馆收据。江阳亮出身份，想从李大北手里拿到名单和照片，没想到李大北狮子开口，直接要价五万。最终江阳向陈明章借钱筹款，约定好交易时间，驱车赶往，怎料胡一浪率先得知此事，立马安排人手放火烧毁照相馆，就连李大北也都丧生在这场大火。金山照相馆门前，两辆车正面相遇，江阳目击胡一浪擦肩驶过，震惊不已。
