# 第23集：周秉昆事业有成买新房

看到不是周秉昆写来的信周志刚心里有些失望，徒弟劝说周志刚主动给周秉昆写一封信，可是周志刚却倔强不肯写，他也想看看周秉昆什么时候能主动给他写信。

宁儿看着周秉昆说话头头是道的样子很是喜欢，还主动热情表示要一直跟随周秉昆的脚步，周秉昆去哪里她就去哪里，吴倩旁边看着宁儿的举动心中有些不舒服。

周蓉回到了贵州，来到了默默的墓地，想起那个夜晚，默默为了她和狼搏斗，最终默默死在了狼的口中，而周蓉得以活命。

姚立松来找周秉义，诉说这些年的不容易，他工作兢兢业业。本想这老厅长升职之后会提拔他，但是没想到老厅长去上任的路上出车祸死了。姚立松也没能如愿的升职，姚立松认为是周秉义成全了周秉昆，但周秉义却告诉姚立松周秉昆不是他成全的。姚立松期盼着郝省长活着，可能很多事都能解决了，但周秉义知道如果省长活着，他都不能在省厅工作。

郝冬梅和周秉义聊天，说起了自己解剖的事情，看到了被解剖的人一切内部器官，本以为他只是一个教具，但是当看见死者手指甲的泥土，才知道他也是一个和大家一样的人。周秉义想起这些年周秉昆的付出和辛苦，突然有些心疼他。

周蓉回到北京之后，发现住处乱七八糟的都是酒瓶子，气得周蓉和冯化成大吵一架。冯化成解释是因为这次的选拔他什么也没有当上，不管是理事还是作协主席都和自己没有关系，周蓉认为北京人才太多了，还不如找邵敬文呢，听周秉昆说过杂志社是求贤若渴，尤其喜欢像是冯化成这样的人才。冯化成虽然不舍得北京的繁华和重心地带，也最终还是答应了。

周秉昆买了一套大房子，让大家一起住进去，郑娟激动掉下眼泪，从来没想过可以住这么大的房子。这里的设施一应俱全，郑娟担心住不长，周秉昆告诉郑娟合同写明了可以住一辈子，这让郑娟更是激动。周母也来到了这个大房子里，拉着周秉昆的手更是激动，从未想过儿子能这么有出息。

晚上，周秉昆起床挨个房间里检查一下，生怕母亲睡不习惯，看着母亲睡着的样子周秉昆这才放心，周秉昆给聪聪盖好被子，聪聪还懂礼貌的感谢父亲的辛苦付出。

1988年，经济飞速发展，骆士宾又上了一个台阶。回到吉春第一件事就是去看看自己的儿子，学校门口，楠楠骑着自行车等着玥玥出来。一个女孩喜欢楠楠希望能乘坐楠楠的自行车，可是被楠楠拒绝了，女孩羡慕玥玥有一个好哥哥。

骆士宾看着楠楠出落得一表人才，也没敢上前打招呼，看着两人骑自行车离开的背影，骆士宾的嘴角露出了笑意。

玥玥看到女孩放在楠楠书包里面的情书，追问楠楠对女孩的看法，楠楠却表示自己并不喜欢那个女孩，女孩虽然长得在班级里算是好看的，但是楠楠并不喜欢。玥玥追问楠楠女孩有没有她漂亮，楠楠笑话玥玥还不如人家，气得玥玥捶打楠楠。

周志刚忽然回来了，但是却发现家里面已经住了国庆和吴倩，国庆打电话告诉周秉昆这件事，周秉昆刚要赶回去的时候酒店里客人却打起来。周秉昆忙着处理店铺的事情没能及时赶回去，国庆只好打电话给了周蓉和周秉义，周蓉将周志刚带去了新房子里，周秉昆生活和工作上都发生了这么大的变化让周志刚很震惊，竟然没有人告诉他一声。

周蓉解释周秉昆一定是想要自己告诉父亲，他们多嘴的话并不好，周志刚也不在多说。当周秉昆揣着忐忑的心情回到家里的时候，周志刚还没有睡觉，一直抽着烟等着周秉昆，他不放心。

听周志刚说不放心自己，周秉昆落泪，为了掩饰眼泪，周秉昆忙躲去了卫生间，周志刚也跟着红了眼眶。调整了情绪之后周秉昆回去房间里，周志刚已经躺下了，周秉昆默默躺在父亲旁边。周志刚心疼周秉昆每天回来这么晚，如果天天这么熬下去生怕身体受不了。周秉昆解释今天接到电话就想赶回来，可是有人打架还惊动了警察，所以处理的比较晚。周志刚责怪周秉昆多年来不写信，周秉昆声称父亲是势利眼，没有出息之前是不能写信的。

周志刚气得将灯打开，想要看着周秉昆的脸说话，周志刚认为攀高踩底的人才是势利眼，而他永远都不是这种人。周秉昆认为周秉义和周蓉考上北大的确他脸上有光，他也的确是想趁着过年带着孩子出去显摆。但是他做错唯一的事情就是没有顾及到周秉昆的感受，让周秉昆没有面子了。

周秉昆突然落泪，质问父亲是否考虑过他的感受？
