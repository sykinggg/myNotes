# 第3集：周秉昆初遇郑娟

蔡晓光希望能让周秉昆去酱油厂工作，因为酱油厂虽然没有这个机械厂体面，但是却很实惠，如果周秉昆去了酱油厂工作，他也算对得起周蓉了。周秉昆思前想后还是答应了，蔡晓光和酱油厂的厂长比较好，答应给周秉昆安排到味精车间，这样干净轻快。

晚上回去的时候，周秉昆在胡同口等着乔海燕回来，主动道歉，并且表示蔡晓光给找了一份工作。乔海燕并未责怪周秉昆，本来还生气呢，但听说周秉昆能去酱油厂，乔海燕还是为周秉昆感到开心。乔海燕再次表达了自己对周秉昆的喜欢，周秉昆却表示自己从未将乔海燕当做女孩子看待，乔海燕也不生气，还认为男人成熟的比较晚，打算慢慢等着周秉昆长大。

酱油厂的厂长接到了蔡晓光的通知就打算安排周秉昆去味精车间，但是遭到了革委会主任曲秀珍的反对，曲秀珍认为不应该走后门长不正之风，因此让安排周秉昆去了最辛苦的出渣车间。

周秉昆第一天上班也算勤恳，但是干了一天之后就觉得很辛苦来找蔡晓光。蔡晓光得知是曲秀珍安排的也表示无奈，因为曲秀珍的丈夫是开国少将，不能得罪，因此劝说周秉昆先干着再说。

乔海燕每天都在路口等着周秉昆回来，还死皮赖脸坐在周秉昆的车后面，胡同里的老街坊都说两人是小两口。周秉昆不愿意接受乔海燕，一看见乔海燕又堵在胡同口，周秉昆调头就往相反的方向走过去，没想到骆士宾和西装男已经等了周秉昆很多天，好不容易等到周秉昆落单将他带走了。

周秉昆也认出了西装男就是那天给涂志强送帽子的人，西装男自称是水自流，涂志强就是因为要帮他出头才打死了人，涂志强临死前说的最后一句话是他媳妇怀孕了。现在涂志强媳妇因为涂志强的死导致没有地方住，还怀着孕。水自流也解释自己不是欺男霸女的坏蛋，有人找麻烦他们躲不掉，才连累了涂志强。而涂志强的老婆因为恨他们连累了涂志强所以不愿意见他们。水自流希望周秉昆能去帮忙看看怀孕的女人，还给了周秉昆三十五元，三十元给女人，五元是跑腿费。随后还约定了时间和地点，每个月都让周秉昆来拿钱。

周秉昆对水自流和骆士宾有些畏惧，哆嗦着接过去钱还有那个女人的地址就骑车离开了。水自流和骆士宾都有些不敢太相信周秉昆，可是目前也只能来找周秉昆。

次日，周母将周秉昆叫起来写信，周秉昆就“埋怨”周母不疼他，就惦记周秉义，周母一边催促周秉昆写信，一边笑嘻嘻给周秉昆喂饭。

周志刚在大三线高道路建设开山凿石，周志刚态度严谨工作认真，因此他所带领的班被嘉奖了一次。周秉昆分别把写好的信都寄出去之后就去找涂志强的妻子，但是纸上的地址不太好找，周秉昆遇到了一个小孩打听涂志强妻子郑娟在哪里，小孩故意不告诉周秉昆。周秉昆正在发愁的时候看到一个老太太拉着糖葫芦车过来不小心跌倒了，周秉昆着急上去帮忙，这才得知刚才的小男孩是郑娟的弟弟双目失明，老太太是郑娟的母亲。

周秉昆见到郑娟的时候，郑娟正坐在炕上穿糖葫芦，因为棉裤拆洗了，所以穿着也比较随意单薄，这让周秉昆看呆了。周秉昆结结巴巴说明自己的来意，郑娟也猜到了是水自流和骆士宾让周秉昆来的。

周秉昆也承认了，还将三十五元钱都拿出来给了郑娟，郑娟非但不要钱，还将周秉昆毫不客气的赶出去。周秉昆骑着自行车离开，郑母跌跌撞撞追出来，不小心滑倒了，周秉昆慌忙掉转头扶起了老太太。老太太祈求周秉昆将钱留下来，因为她一个人养活不了一家人。

老太太告诉周秉昆郑娟和涂志强并未结婚，但是却怀孕了，现在户口能不能上都是一个问题，她实在不知道将来还能如何面对这一切的困难。周秉昆将钱给了老太太，老太太抱着钱回去了。郑娟弟弟并未回去，而是拉着周秉昆询问她是否哭了？询问周秉昆还会来吗？
