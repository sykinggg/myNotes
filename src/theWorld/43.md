# 第43集：楠楠出意外身亡

玥玥站在门外，将周蓉吓了一跳，周蓉责怪玥玥没良心，如果不是楠楠发消息她就不回来。玥玥调侃周蓉，认为自己比周蓉当年跑去贵州强，周蓉尴尬看向蔡晓光。蔡晓光却并不理会，笑言，自己才是笑到了最后的人，蔡晓光贴心为娘俩做了她们爱吃的饭菜，一家人和睦快乐。随后一家三口还特意去看望周秉昆，两家人聚在一起畅聊很久。

玥玥还特意来看了郝母，离开的时候郝母对玥玥充满不舍，玥玥突然跑回来抱着郝母，郝母露出开心的笑。与其说这些年郝母照顾了玥玥，郝母倒是认为是玥玥陪伴了她。

六君子一年一度的相聚，商议着要将孩子们都带着，以后孩子们也和他们一样，做生意干工作都是有来有往相互帮助。众人闲聊了自己生活的不如意和如意，随后又一起去看了曲秀珍，曲秀珍告诉周秉昆改革就是要有人背负骂名的，周秉义就是为祖国背负的骂名，这也是为什么他会被提拔的原因。

周秉义被调去哈阳市当市长，这让郝冬梅不满，认为吉春是省会。虽然哈阳市是市长，可是级别和吉春的副市长差不多。郝母讲了一些干部任用的事情，但郝冬梅依然无法理解，周秉义劝说郝冬梅放心，市长毕竟办事容易，人都是宁可当鸡头也不当凤尾。经过劝说之后，郝冬梅这才放了周秉义离开，但也充满了不舍，周秉义认为郝冬梅在吉春的工作不错，也不希望她跟着自己一起来回调。

周秉义刚去哈阳市的时候，大家都举着牌子拦着路，咒骂着周秉义，让周秉义赶紧回去。到了哈市之后，周秉义召开了领导班子的会议，了解当地的情况，也认识了肖家河副市长。在那个特殊的年月里，就是有人急于甩锅想要离开，有些人却忍辱负重砥砺前行，而周秉义就属于后者。

由于哈阳市工人上访，省里召开会议的时候周秉义被点名批评了，周秉义不想再干市长，他想要去做教育。而周秉义去了哈市工作也开展不起来，表面上叫着市长，可是却没人听他的，都骂着他就会批条子砸工人的饭碗。周秉义将这件事告诉了郝母，郝母认为周秉义遇到困难不该退缩，不可否认周秉义在吉春的时候干得非常好，现在去了之后就不能听到骂声就不干了，正应该杀出一条血路，将那些骂声置之脑后。

郝母告诉周秉义，他是刚刚被调去当正市长，如果想要申请调去高校是没有人会同意的，除非走后门。郝母也答应帮忙调动，可周秉义却临时改变了主意，他打算继续下去，让郝母向郝冬梅保守这个秘密。郝母笑言理解周秉义想要在媳妇面前保持男子汉形象。郝冬梅看到周秉义回来激动抱着他，夫妻俩难得相聚，郝母给两人留出了空间。

向阳是化工研究所的人，骆士宾希望能将他聘请过来，可是向阳舍不得研究所的工作。骆士宾提出让他请假三个月帮着筹备化工厂的事情，但是向阳还是拒绝了，无奈之下骆士宾找到了周秉昆，周秉昆理解一个正规单位的人心里的想法，答应帮着劝说。至于楠楠放假回来的事情，骆士宾希望楠楠能去他那里，周秉昆答应会说服郑娟，但是楠楠必须得留出时间来陪着郑娟，骆士宾自然满口答应。

楠楠给玥玥写信，他会提前一年的时间来完成研究生学业，回到家里陪着父母和玥玥。双方约定了时间之后让玥玥提前去家里等着。全家人都盼望着楠楠回来，在朋友的帮忙下开车打算去机场，不成想中途车坏了，这时突然停下来一辆车下来两人，要借他们的钱加油，楠楠主动拿出钱包多给，没想到对方抢走了楠楠所有的钱，而且还要求楠楠的朋友也交出所有的钱。朋友不肯，对方拿出手抢对准了楠楠的朋友，楠楠挺身护着朋友，却因此丧命。

全家人等了很久没有见到楠楠回来，接到学校电话的蔡晓光赶来告诉了这个噩耗。玥玥抱着她和楠楠的合影放声大哭，郑娟不吃不喝一直躺着，周秉昆安慰已经不起作用，蔡晓光打算去找郑光明。蔡晓光握着周蓉的手，鼓励周蓉要坚强，因为玥玥只有她了。郝冬梅也将这个消息告诉了周秉义。

周秉昆一直强忍着，看到周秉义的时候终于忍不住哭出声来。
