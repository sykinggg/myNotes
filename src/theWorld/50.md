# 第50集：周秉昆被释放出狱

曹德宝拉着郑娟来找龚维则，但龚维则却一推再推，声称现在的洗浴中心不在自己的管辖范围内无法管，还建议让郑娟去找周秉义，只要周秉义一个电话问题就解决了。可郑娟当即反对，她知道周秉义的为人，这种事找了也没用，气得曹德宝摔门离开。

姚立松发现彭心生现在的公司在做房地产很惊讶，但是彭心生表明要跟着时代的发展，要是骆士宾在的话也不想不到如今房地产的发展的趋向那么好，如果合作的话，彭心生打算分三成给姚立松，但是姚立松却表示他想要原始股。和骆士宾公司合作的有一家东北企业，水自流让彭心生安排玥玥前去接洽，因为那家公司能做的那么好，和周秉义的主抓脱不了关系，而玥玥又和周秉义关系特别。

周蓉听说章早坡要去吉春出版社工作心里很难受，也想起了蔡晓光的话，当初就是因为章早坡说过要去贵州教书，她才选择了章早坡。当时蔡晓光就说过，想要去完成周蓉的梦想，不是上下嘴唇一碰那么简单的事情，意有所指这个章早坡也不会替周蓉完成梦想。现实给了周蓉重重一击，周蓉特意叫了章早坡问话，因为去出版社也需要周蓉的评价。章早坡很紧张，讲述了自己家里的穷苦，想要留下教书被村子里和家里人骂，都觉得他是疯了，他们都需要生活。周蓉最后还是给了章早坡很好的评价，这让章早坡有些意外，但也很感动。

郑娟找曹德宝解释自己并不是不想帮忙，而是因为周秉义真的不帮忙，特别是周秉昆的事情他都没帮忙，更别说别人了。曹德宝觉得郑娟就是虚头巴脑的人，不务实，当即就把郑娟赶了出去。玥玥回来看郑娟，得知这件事之后，玥玥告诉郑娟，以后如果有人要帮忙，就当着他们的面打电话，周秉义拒绝了让他们都听听，也不是你不帮忙，郑娟惊讶于玥玥现在竟然如此会办事。玥玥认为乔春燕和周秉昆毕竟关系好，所以打算找律师帮忙给救出来。

洗浴中心的梅子拿着一个账本来找曹德宝，上面记录着秦守德送礼的来往记录，只要这个本子交给龚维则，龚伟则天涯海角也会将秦守德抓回来，秦守德能提前逃走必然也是有人通风报信所致。这件事果然奏效，乔春燕很快就被放出来了，还亲自带着曹德宝去道歉，乔春燕认为如果不是郑娟跟着去派出所，龚维则都不会去见曹德宝，更何况玥玥还要找律师帮忙。

乔春燕想买房子，手里缺钱，就找到龚维则借钱，还特意说了玥玥要帮着打官司的事情，以此证明和周秉义的关系，龚维则只好乖乖借了五万块给乔春燕，乔春燕表面答应一定还钱，可是龚维则知道这笔钱是不会还的。乔春燕很快买了房子，一家人开心幸福。

2001年，郝母生病住院，作为过去的老领导，省里领导特意来探望，郝母趁着领导来的时候提出了一个要求。周秉昆被判刑九年，因为表现良好两次减刑，原本是大年初一该释放回家，可是过年行政人员放假，可能就需要过完年才能办手续。因此，郝母希望领导能协调，让周秉昆提前一天回家过年，赶上春节的团聚。领导立刻表示会着手调查，随后让人调查周秉昆的情况，如果符合减刑的条件，就让他回去。办完这件事，郝母永远闭上了眼睛，周秉义的愿望希望自己死了之后也没有任何的追悼会。

周秉昆走出监狱的大门，周蓉、周聪和玥玥来接他，还特意为周秉昆买了新衣服，公司还专门给玥玥配备了新车。坐在车上，周秉昆看着外面的变化，有些不敢相信变化竟然那么大，高楼大厦让人眼晕。

马上过年了，孙胜也在房间里拉扯上彩灯，于虹伤感落泪，孙胜哄着于虹，希望于虹能照顾好身体，因为有妈的孩子不管多大都有孩子气，可是没有妈妈就好像鲜花插在瓶子里，虽然有香气，是没有根的。周聪领着周秉昆泡了个澡，周秉昆从周聪这里得知赶超卧轨的事情非常震撼，满脑子都是赶超的身影。

孙小宁突然回来了，于虹抱着孙小宁激动落泪。
