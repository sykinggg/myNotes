# 第51集：周秉昆出狱回家

孙小宁带回来了钱，希望孙胜能好好读书，这些年她去了深圳的电子厂上班，利用业余时间去读了夜大，每天休息也就三四个小时，但孙小宁觉得很充实。前几年就想回来的，但是总怕给家里丢脸，所以就一直没有回来。孙小宁叮嘱孙胜好好读书，家里好不容易出来一个大学生，他读到哪里孙小宁就供到哪里。孙小宁的回归，让于虹的心里也有了一线曙光。

周秉昆带着包推开了房门，郑娟在房间里默默等着，回头看见周秉昆，郑娟的眼睛里噙满了泪水，一句回来了道不尽对周秉昆的思念。周蓉和玥玥没有一起回来，想要给周秉昆和郑娟单独相处的空间。郑娟告诉周秉昆年三十都来这里过年，都陪着周秉昆一起过，周秉昆回头看向郑娟，轻轻嗯了一声，将郑娟拥入怀中。

于虹带着孙小宁来到乔春燕家里，家中吴倩和国庆也都等在这里，吴倩抱着孙小宁潸然泪下。周秉昆和郑娟来到门外，看着门口贴着对联，横批就是欢迎周秉昆回家，门打开了，乔春燕看见周秉昆抱着他泪如雨下。

周秉昆看着一张张熟悉的脸，眼泪在眼眶里打转，每个人都激动到哽咽，孙小宁站在人群后，看着周秉昆，眼眸中的泪水不自觉落下。周秉昆看到孙小宁，想起了那个辛苦劳作到最后卧轨而死的赶超，此时的周秉昆肝肠寸断。小十年未见了，每个人还都一如既往的亲切，浓浓的亲情笼罩着每一个人。

孙晓光和编剧启文谈剧本的事情，让他不要着急慢慢修改，可启文却告诉蔡晓光他已经将剧本卖给了一家大公司。启文告诉蔡晓光卖给他们不是因为钱，是因为之前蔡晓光的资金链断裂了，他的剧本可以说是十月怀胎，不能有丝毫的闪失。蔡晓光愤怒斥责启文不早说，启文无奈表示也是前天收到的合同。蔡晓光提出再加三十万，让启文先将那边给推了，可是启文却表示合同已经签订了，违约金他赔不起。

松江市，周秉昆视察工作离开的时候却被一个女人拦住了车，女人提出要找市里的领导，一听说周秉义是中央部来的干部，立刻冲上去，因为她之前写了很多信都石沉大海了。周秉义耐心听取女人娓娓道来。

因为好几个当官的拉着妇女的女儿去喝酒，还非礼她的女儿，在她女儿夺路而逃的时候跌落酒店的楼梯摔伤了，至今昏迷不醒。酒店和当官的都推辞没有责任，当官的给了一万元还说是人道主义精神才给的，妇女的女儿现如今在医院昏迷不醒，面临停药，她实在没有办法了写信去市里举报，可是都没有结果。无奈之下只好来到市委想要求救，恰好碰到了周秉义。

由于周秉义不能回来，玥玥吞吞吐吐将这件事告诉了周秉昆，郑娟慌忙解释，肯定是因为周秉义事情多，所以被绊住了。至今周蓉也没有来，玥玥要去催促，周秉昆却拦住了，心里虽然不舒服，可是也知道都是忙人。

周蓉给蔡晓光打电话，想要约好去找周秉昆，却怎么也找不到人，只能来他的单位找，助理告诉周蓉因为剧本的事情，蔡晓光心情不好，将自己关在房间里很长时间，谁也不许打扰。周蓉就蹲在门外，也不去打扰。

周秉义知道只要他走了，那个女孩可能就活不了了，因此就留下没有回去。和松江市的市委领导沟通，这几个干部当中史清泉是职位最高的，周秉义提出从他着手调查，但是市里领导认为史清泉是抓经济的专家，也是骨干，现在不能先调查。周秉义提出了自己调查之后的情况，在几条繁华的商业街都被这里的百姓称之为腐败街。市委的人提出等过完年再说，可是周秉义却打算就地过年，因为对于百姓来说，年关难过。

当蔡晓光打开门的时候，看到周蓉蹲在门外，整个办公楼早已没有人了，大家也都回家过年去了。蔡晓光告诉周蓉，他已经组织了剧组，但是编制却将剧本卖出去了，现在他觉得没脸了，只能解散剧组。周蓉提议去找找那家买剧本的公司，说不定就可以让蔡晓光去导演，但蔡晓光却不同意，认为主动去找别人和别人找他是两码事。周蓉自责不已，蔡晓光却反过来安慰周蓉。

蔡晓光知道周秉昆回来，拉着周蓉的手要跟着一起去，还声称想周秉昆了，他刚回来，一家人必须去。蔡晓光笑言以后要靠周蓉养活了，周蓉甘之如饴，但却心情沉重，自责连连。

郝冬梅责怪周秉义不该不回去，毕竟周秉昆是刚出来，肯定也会很失望。周秉义担心那个女孩的生命安全，纪委和市里都收到过匿名信，可是都没有去落实查证，这里面的事情可想而知。

郝冬梅回来之后，特意给周秉昆带了一身衣服，关心周秉昆接下来的安排。郝冬梅告诉周秉昆因为母亲去世，省委的房子要收回，她就要搬去医院的福利房居住，周秉昆和郑娟都表示要去帮忙搬家。

吴倩书店也要黄了，书店的书都要一折卖，因此就找了周秉昆过来帮忙运输，国庆此时也已经从军工厂退休了，总觉得自己年轻一把子力气没地方用，几个人一起骑着三轮车拉着平板车全都去搬书赚钱了。

经过这件事之后，周秉昆和郑娟提出要买个车，因为现在拆迁的人特别多，所以肯定会不少搬家的人，他想要成立一个搬家服务公司，到时候搬家收拾就都有了。这些年郑娟攒下了六千多。可周秉昆觉得相差很远，最起码也得六万。

周蓉来找了光宴影视公司的经理马丽丽，想要谈一下蔡晓光导演的事情，周蓉也看了剧本，觉得很难得。周蓉希望马丽丽能让蔡晓光来导演这部剧，但却从马丽丽这里得知严总已经交给了王野导演。周蓉临走时候听到马丽丽接了一个电话，她孩子想去省实验学校。

周蓉回去的时候，蔡晓光正在看剧本，他想要找一个不错的剧本，这样就不用解散剧组了。但剧本的低劣又不符合他的要求，蔡晓光觉得这种剧如果拍出去丢人，周蓉提出让蔡晓光帮忙给朋友的孩子找找省实验学校的关系，蔡晓光难得看见周蓉为朋友说话，立刻就答应托关系找人。
