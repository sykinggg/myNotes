# 第58集：周秉昆郑娟牵手一生

大家唱着《朋友》喝着杯中的酒，于虹、吴倩和郑娟相互搂着开怀大笑，曹德宝和乔春燕不自觉的往前走，透过橱窗看着里面的情形，往事一幕幕浮现在脑海中，乔春燕和曹德宝泪如雨下，乔春燕自觉没有脸进去含泪离开。

乔春燕想要把房子给牛牛，牛牛也表示女友不要房子，只要两人在一起就行，现在登记就可以了。牛牛也知道了曹德宝举报信的事情，认为父母俩就是太精了，太会算计。在牛牛看来于虹是真的太难了，可是乔春燕就是嫉妒人家。乔春燕也意识到自己的错误，牛牛告诉乔春燕他在殡仪馆工作被人瞧不起，但是他就认为自己是一身正气，就好比当年的乔春燕，貌美如花却可以去给人修脚。他像极了乔春燕，乔春燕被牛牛说得落泪。

周秉义回到家里，兄弟姐妹早已等候在家，欢迎他的回归。

乔春燕和曹德宝来找国庆，希望能参加牛牛的婚礼，国庆不满二人的所作所为，将曹德宝和乔春燕训斥了一顿，二人悻悻离开。回去之后，曹德宝和乔春燕开始准备请柬和礼物，曹德宝还特意提前演讲了一段新婚祝福。但想到不会有人来了，乔春燕黯然神伤，可没想到敲门声响起，来的人正是周秉昆和郑娟。

牛牛结婚这天，很热闹，郑娟和周秉昆还有大家都来祝贺，乔春燕还是和以前一样，抱着周秉昆送上了一个脸颊吻，郑娟站在一边笑看二人。

吴倩和国庆要去深圳找儿子大刚，大刚在职读了研究生，在单位很受器重，现在非要父母过去，吴倩嘴上不舍朋友，但满脸洋溢骄傲和幸福。于虹感谢所有人对自己的帮助，也相信赶超如果泉下有知也会感谢大家的帮助，周秉昆称赞于虹养了一个好儿子，孙胜那么年轻就做了教授。每个人的孩子都有了出息，只要根正，无论是否考上大学，相信都差不了。

郝冬梅和周秉义给周秉昆的孙子起名字，因为周秉昆说要让孙子管周秉义夫妇叫爷爷奶奶，郝冬梅觉得这是在弥补两人的遗憾，周秉义笑言应该取一个简单的名字，要不然周秉昆该不乐意了，小时候周秉昆因为名字复杂还非要和周秉义换名字。

周蓉用了三年时间写了一部四十集的电视剧发给了一个影视公司，但是没想到兜兜转转竟然到了蔡晓光这里，蔡晓光认为剧本不错，但是因为第一次写结构有很多问题。但蔡晓光愿意接下来，总觉得有似曾相识的感觉，周蓉这才说出这是她写的。周蓉希望蔡晓光能如实说出看法，她宁可被拒绝也不要施舍，蔡晓光认为剧本距离他的要求还差着一截子，但是他要好好帮着周蓉改改，作为自己的封箱之作。

周秉义退休了，和郝冬梅去了当年插队的地方，回忆往昔，手拉手走过当年的吊桥。当年的地方现在已经成为了旅游景点，恰好姚立松也在这里，透过玻璃窗看到了外面欣赏美景的周秉义和郝冬梅，姚立松颤抖着过去呼唤着二人，退休了都觉得寂寞了。

周秉义和姚立松聊天，姚立松好奇这些年周秉义是如何做到从政几十年都能干干净净的。周秉义表示自己怕死不瞑目，他害怕没有脸去见爹娘，害怕同学戳脊梁骨，周秉义忽然劝说姚立松去自首，姚立松沉默了。当看见郝冬梅给周秉义拿来一盒子药的时候，姚立松以为是周秉义养生，却没想到周秉义哈哈一笑告诉姚立松他是胃癌晚期。周秉义告诉姚立松家里人也都知道这件事，想要在生命结束之前出来走走，大家也都同意了。姚立松闻听此言忽然哭了，来到这里，每次都能想到挥洒青春的时候，姚立松也早就准备好了材料，只想回去就自首，两个老友握手道别。

蔡晓光和周蓉也一起来到了贵州金坝村小学，学生们跳起了篝火舞，看着大家载歌载舞的样子，周蓉像是孩子一般笑了，拉着蔡晓光的手感谢他，两人一起投入到了跳舞的队伍中，欢声笑语不断。

郝冬梅和周秉义坐在大山之巅，此时的周秉义环顾四周牛羊成群，身边坐着郝冬梅。周秉义写了两封信，分别给了周蓉和周秉昆，周秉义一生最大的幸运就是遇到郝冬梅，如果他有一天不在了，希望周秉义和周蓉能代替他照顾好郝冬梅。周秉义叮嘱弟弟和妹妹，要好好生活，可以不做很成功的人，但是必须做一个爱国的人。

周秉昆撑着伞和郑娟漫步雨中，他期望还能有来生，还能和郑娟在一起，郑娟嘴上笑着周秉昆想得美，周秉昆嘴角扯出一丝笑意，想想来生能在一起他就心里美，郑娟依偎在周秉昆身上憧憬未来。
