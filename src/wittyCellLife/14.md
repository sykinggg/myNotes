# 第14集：金济赫治疗心理疾病 刘汉阳险遭人陷害

金民哲的假释申请马上就要审批了，最近因为假释的事情，他一直小心谨慎的生活。彭部长告诉金民哲，监狱给他安排了毕业生的采访。本以为金民哲不会答应，却没有想到金民哲略思索了一下之后，便答应了。来采访金民哲的是一个年轻又充满活力的年轻大学生，她性格活泼开朗，很快便打开了话匣子。金民哲也从她的身上，感受了监狱所没有的朝气。他开始庆幸自己答应了采访，并开始期待女孩的到来。

刘汉阳发现了金济赫的投球有问题，他无法投直线的球。李俊浩偷偷为金济赫找来了心里医生，证实他患上了布拉斯综合症。可是几次三番治疗试下来，金济赫的情况却没有丝毫好转，李俊浩找到了智孝。希望她能在金济赫的棒球生涯结束时，能来见见他。智孝告诉李俊浩，他将金济赫想的过于复杂了。对于这种单纯的人，最直接的方式就是最好的方式。按照智孝的方法，金济赫成功的治好了心理疾病。

6号房的犯人们，大部分都到了刑期的最后段。安东浩提醒他们，现在应该安分生活。他觉得2舍栋的那些黑帮份子的首要目标应该是家庭宽裕的刘汉阳。刘大蔚却在这时告诉他们，他的哥哥为了他的事情已经被校方开除了。而他居然一直瞒着自己，如果不是这次律师说漏了嘴，他可能还蒙在鼓里。

西部监狱爆发了监狱毒品的丑闻，为了挽回形象。金所长命令罗科长一定要将这些人全部挖出来，罗科长接到命令后，为了显示自己的能力。强行让瘾君子多加二个嫌疑人上去，而瘾君子们，为了能够减刑，将目标定在了病殃殃的刘汉阳身上。

为了给刘汉阳服下药，他们将惩戒室的所有犯人包括自己全部下了药。

而另一边安东浩告诉他们，瘾君子们突然全部进入了医务室。彭部长知道后，顿感不妙。当他赶到时，宋担当早已经将人制服带走了。而刘大蔚还拿着筷子站在门口，原来，宋担当为了答谢刘大蔚昔日的救命之恩，特意请刘大蔚吃炸机。这时宋担当的昔日同学，现在的医务室医生也过来了。并无意中透露了，医务室只有瘾君子在，这让军人出生的刘大蔚感觉到了不同寻常。

刘汉阳因为刘大蔚救了自己一次，而开始不在针对他。刘汉阳知道刘大蔚正在烦恼什么，他告诉刘大蔚，他应该尊从自己的内心去生活。明明心里就希望哥哥不要放弃自己，陪自己走完这再审的这段路。那为什么还在假装的让哥哥不要管自己呢？刘汉阳的话，打开了刘大蔚的心门。他勇敢的告诉哥哥，希望哥哥能够帮帮他。
